AILmanac शून्य में मौजूद नहीं है, और हम हर चीज़ को प्रतिस्थापित करने का दिखावा नहीं करेंगे। यहाँ सर्वोत्तम संसाधनों का एक ईमानदार मानचित्र है — और इन्हें हमारे **बजाय** कब उपयोग करें।

## सत्य का स्रोत

- **[Anthropic दस्तावेज़](https://docs.anthropic.com)** — Claude, API, मॉडल और सुविधाओं के लिए आधिकारिक संदर्भ। **जब AILmanac और दस्तावेज़ में असहमति हो, तो दस्तावेज़ जीतते हैं।** सटीक, वर्तमान विवरण के लिए उनका उपयोग करें; विचारशील, स्तरबद्ध, करके-सीखने वाले मार्गदर्शन के लिए हमारा उपयोग करें।

## आधिकारिक Anthropic संसाधन

- **[Prompt Library](https://docs.anthropic.com/en/resources/prompt-library/library)** — तैयार उदाहरण प्रॉम्प्ट।
- **[Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)** — चलाने योग्य API रेसिपी (टूल उपयोग, RAG, विज़न)।
- **[Claude Code दस्तावेज़](https://docs.anthropic.com/en/docs/claude-code/overview)** — CLI/IDE संदर्भ।
- **[Anthropic Academy / Learn](https://www.anthropic.com/learn)** — मुफ़्त मार्गदर्शित पाठ्यक्रम।
- **[Model Context Protocol](https://modelcontextprotocol.io)** — खुला MCP मानक + सर्वर निर्देशिका।

## समुदाय और सामान्य

- **[awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code)** — चयनित कमांड, हुक और वर्कफ़्लो।
- **[Prompt Engineering Guide](https://www.promptingguide.ai)** — गहन, विक्रेता-तटस्थ प्रॉम्प्टिंग संदर्भ।
- **[Learn Prompting](https://learnprompting.org)** — संरचित, शुरुआती-अनुकूल प्रॉम्प्टिंग पाठ्यक्रम।

## क्या कब उपयोग करें

| आप चाहते हैं… | यहाँ जाएँ |
|---|---|
| सटीक वर्तमान API/सुविधा विवरण | आधिकारिक दस्तावेज़ |
| कॉपी-पेस्ट उदाहरण प्रॉम्प्ट | Prompt Library / हमारे [टेम्पलेट](/docs/templates/prompts) |
| चलाने योग्य API कोड | Cookbook |
| एक विचारशील, स्तर-चिह्नित पाथ | **AILmanac** |
| प्रॉम्प्टिंग पर एक गहन सैद्धांतिक संदर्भ | promptingguide.ai |
| एक मार्गदर्शित पाठ्यक्रम | Anthropic Academy |

## AILmanac कहाँ फिट बैठता है

हम **विचारशील फ़ील्ड गाइड** हैं: पहले एक अनुशंसित तरीका, स्तर के अनुसार चिह्नित, टेम्पलेट-समृद्ध, ताज़गी के बारे में ईमानदार, और Claude **तथा** सामान्य AI अवधारणाओं को कवर करने वाला। हमें उस मानचित्र की तरह समझें जो आपको सही समय पर सही गहन संसाधन की ओर इशारा करता है — और बुनियादी बातें सिखाता है ताकि वे सब समझ में आ जाएँ।
