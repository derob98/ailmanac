वह ट्रैक चुनें जो आपसे सबसे ज़्यादा मेल खाता हो। हर एक एक क्रमबद्ध पाथ है — इन्हें क्रम में करें, जो आप पहले से जानते हैं उसे छोड़ दें। समय अनुमानित हैं।

## 🌱 जिज्ञासु नवागंतुक · ~45 मिनट

आप प्रोग्रामर बने बिना AI का अच्छा उपयोग करना चाहते हैं।

1. [आपके पहले 5 मिनट](/docs/start-here/your-first-5-minutes) <LevelBadge level="beginner" />
2. [मुझे कौन-सा Claude उपयोग करना चाहिए?](/docs/start-here/which-claude) <LevelBadge level="beginner" />
3. [प्रॉम्प्टिंग की बुनियादी बातें](/docs/prompting/basics) <LevelBadge level="beginner" />
4. [LLM क्या है?](/docs/foundations) <LevelBadge level="beginner" />
5. [रोज़मर्रा की उत्पादकता](/docs/playbooks/productivity) <LevelBadge level="beginner" />

## 💼 ज्ञान कार्यकर्ता · ~2 घंटे

आप चाहते हैं कि AI आपके वास्तविक काम को तेज़ बनाए — लेखन, शोध, विश्लेषण।

1. जिज्ञासु नवागंतुक ट्रैक (ऊपर), फिर…
2. [प्रॉम्प्ट पैटर्न लाइब्रेरी](/docs/prompting) <LevelBadge level="all" />
3. [भ्रम (हैल्यूसिनेशन) और उन्हें कैसे कम करें](/docs/foundations) <LevelBadge level="intermediate" />
4. आपकी प्लेबुक: [लेखन](/docs/playbooks), [शोध](/docs/playbooks), या [डेटा विश्लेषण](/docs/playbooks)
5. [प्रोजेक्ट्स: स्थायी कार्यस्थल](/docs/claude-app) और [पुन: प्रयोज्य प्रॉम्प्ट टेम्पलेट](/docs/templates)

## 🛠️ बिल्डर / डेवलपर · ~आधा दिन

आप कोड लिखते हैं और Claude Code को अनुकूलित करना तथा API पर निर्माण करना चाहते हैं।

1. [Claude Code क्या है](/docs/claude-code/what-is-claude-code) <LevelBadge level="beginner" />
2. [CLAUDE.md और मेमोरी फ़ाइलें](/docs/claude-code) <LevelBadge level="beginner" /> + [प्लान मोड](/docs/claude-code)
3. [अनुमतियाँ](/docs/claude-code) और [settings.json](/docs/claude-code) <LevelBadge level="intermediate" />
4. [आपका पहला API कॉल](/docs/api/first-call) → [टूल उपयोग](/docs/api) → [एजेंट बनाना](/docs/api) <LevelBadge level="advanced" />
5. [वॉकथ्रू: एक वास्तविक रिपॉज़िटरी के लिए Claude Code को अनुकूलित करें](/docs/walkthroughs) <LevelBadge level="intermediate" />
6. [सुरक्षा और ज़िम्मेदार उपयोग](/docs/security) <LevelBadge level="advanced" />

## 👥 टीम लीड · ~3 घंटे

आप एक टीम में AI को रोल आउट कर रहे हैं और संगति, सुरक्षा और लागत की परवाह करते हैं।

1. बिल्डर ट्रैक की बुनियादी बातें (ऊपर 1–3)
2. [प्लगइन्स और मार्केटप्लेस](/docs/claude-code) — एक साझा टूलकिट शिप करें <LevelBadge level="advanced" />
3. [लागत और लेटेंसी समझौते](/docs/foundations) और [एक मॉडल चुनना](/docs/api/choosing-a-model) <LevelBadge level="intermediate" />
4. [एजेंट को सुरक्षित करना](/docs/security) और [स्वायत्त रन को सुदृढ़ करना](/docs/security) <LevelBadge level="advanced" />
5. [ज़िम्मेदार उपयोग और सत्यापन](/docs/security) — टीम के मानदंड तय करें <LevelBadge level="all" />

---

तय नहीं कर पा रहे कि कौन-सा सही है? [आपके पहले 5 मिनट](/docs/start-here/your-first-5-minutes) से शुरू करें — आप चाहे कहीं भी पहुँचें, यह उपयोगी है। और [स्तर और ताज़गी कैसे काम करते हैं](/docs/start-here/levels-and-freshness) पढ़ें ताकि बैज और तारीखें समझ में आएँ।
