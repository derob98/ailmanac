यह पेज एक चलता हुआ, मनुष्य-द्वारा क्यूरेट किया गया सारांश है। बदलावों की प्रामाणिक, पूर्ण सूची के लिए, हमेशा आधिकारिक Anthropic डॉक्स और Claude Code चेंजलॉग देखें।
</VerifyNote>

Claude में हाल ही में क्या बदला है इसका सरल भाषा में सारांश — ताकि आप हर चेंजलॉग पढ़े बिना "क्या नया है और इसका मेरे लिए क्या मतलब है" को सरसरी नज़र से देख सकें। नवीनतम पहले।

:::info यह पेज कैसे वर्तमान रहता है
प्रविष्टियाँ समुदाय द्वारा और एक [स्वचालित वॉचर](/docs/contribute/automation) द्वारा जोड़ी जाती हैं जो किसी मनुष्य के लिखने हेतु अपस्ट्रीम बदलावों को फ़्लैग करता है। प्रत्येक प्रविष्टि अपने आधिकारिक स्रोत से लिंक करती है। कुछ छूटा हुआ दिखता है? [इसे जोड़ें](/docs/contribute/contribute-in-10-minutes)।
:::

## एक प्रविष्टि कैसे पढ़ें

प्रत्येक अपडेट को **सरफेस** के अनुसार टैग किया जाता है — 🗣️ Apps (चैट/वॉइस/मोबाइल), 🛠️ Claude Code, 🔌 API, 🧠 Models — और "**इसका आपके लिए क्या मतलब है**" का उत्तर देता है।

## हालिया अपडेट

- 🧠 **Models** — **Claude Sonnet 5** 30 जून, 2026 को शिप हुआ और अब यह **Claude Code में डिफ़ॉल्ट मॉडल** है, जिसमें एक नेटिव 1M-token कॉन्टेक्स्ट विंडो और 31 अगस्त, 2026 तक परिचयात्मक मूल्य निर्धारण है। यह नया संतुलित "यहाँ से शुरू करें" Sonnet टियर है, जो Claude Sonnet 4.6 (अब एक लीगेसी मॉडल) की जगह लेता है। *इसका मतलब:* यदि आप किसी मॉडल को पिन करते हैं, तो `claude-sonnet-5` पर माइग्रेट करें; और किसी ब्लॉग पोस्ट के आँकड़े पर निर्भर रहने से पहले हमेशा सटीक IDs और लाइव मूल्य निर्धारण के लिए [वर्तमान मॉडल और मूल्य निर्धारण](/docs/whats-new/models-and-pricing) तालिका देखें। <VerifyNote lastVerified="2026-07-01" source="https://platform.claude.com/docs/en/about-claude/models/overview">API ID `claude-sonnet-5`; Sonnet टियर; 1M-token कॉन्टेक्स्ट; 128k अधिकतम आउटपुट; Jan 2026 नॉलेज कटऑफ। $2 / $10 प्रति MTok का परिचयात्मक मूल्य निर्धारण 31 अगस्त, 2026 तक लागू है (मानक $3 / $15)। अब [changelog](https://code.claude.com/docs/en/changelog) के अनुसार Claude Code डिफ़ॉल्ट (v2.1.197, 30 जून 2026)।</VerifyNote>
- 🛠️ **Claude Code** — **Dynamic workflows** रिसर्च प्रीव्यू में हैं: इसके लिए अनुरोध करें (या `/effort ultracode` सेट करें) और Claude एक JavaScript स्क्रिप्ट लिखता है जो पृष्ठभूमि में दसियों से सैकड़ों [सबएजेंट्स](/docs/claude-code/subagents) को व्यवस्थित करती है — कोडबेस-व्यापी ऑडिट, बड़े माइग्रेशन और क्रॉस-चेक किए गए शोध के लिए। बंडल किया गया देखने के लिए `/deep-research` चलाएँ, या रन देखने के लिए `/workflows`। *इसका मतलब:* अब आप ऐसे फ्लीट-स्केल कार्य कर सकते हैं जिन्हें कोई एक बातचीत कभी समन्वित नहीं कर सकती, और यह व्यवस्था एक दोबारा चलाने योग्य कमांड के रूप में सहेजी जाती है। देखें [Dynamic Workflows & ultracode](/docs/claude-code/dynamic-workflows)। <VerifyNote lastVerified="2026-06-24" source="https://code.claude.com/docs/en/workflows">Claude Code v2.1.154+ आवश्यक है; पेड प्लान्स, API, Bedrock, Vertex और Foundry पर उपलब्ध।</VerifyNote>
- 🧠 **Models** — एक नया टॉप-टियर मॉडल, **Claude Fable 5**, जून 2026 में शिप हुआ (Claude Code v2.1.170+)। *इसका मतलब:* किसी मॉडल को पिन करने से पहले लाइव लाइनअप और सटीक IDs के लिए [वर्तमान मॉडल और मूल्य निर्धारण](/docs/whats-new/models-and-pricing) तालिका देखें — किसी ब्लॉग पोस्ट से मॉडल संबंधी तथ्यों को कभी हार्ड-कोड न करें।
- 🛠️ **Claude Code** — सबएजेंट्स अब **अपने स्वयं के सबएजेंट्स बना सकते हैं** (5 स्तर तक गहराई में), और `/code-review --fix` समीक्षा निष्कर्षों को आपके वर्किंग ट्री पर लागू करता है। *इसका मतलब:* डेलिगेशन अधिक गहराई तक संयोजित होता है, और समीक्षा केवल रिपोर्ट करने के बजाय लूप को बंद कर सकती है। सटीक व्यवहार के लिए [आधिकारिक चेंजलॉग](https://code.claude.com/docs/en/changelog) से क्रॉस-चेक करें।
- 🔌 **API** — संरचित आउटपुट, विस्तारित सोच, और प्रॉम्प्ट कैशिंग जैसी क्षमताएँ विकसित होती रहती हैं। *इसका मतलब:* सटीक पैरामीटर पर निर्भर रहने से पहले [API अनुभाग](/docs/api/first-call) को फिर से जाँचें।
