Choisissez la voie qui vous ressemble le plus. Chacune est un parcours ordonné — suivez-les dans l'ordre, sautez ce que vous connaissez déjà. Les durées sont approximatives.

## 🌱 Le novice curieux · ~45 min

Vous voulez bien utiliser l'IA sans devenir programmeur.

1. [Vos 5 premières minutes](/docs/start-here/your-first-5-minutes) <LevelBadge level="beginner" />
2. [Quel Claude devrais-je utiliser ?](/docs/start-here/which-claude) <LevelBadge level="beginner" />
3. [Les bases du prompting](/docs/prompting/basics) <LevelBadge level="beginner" />
4. [Qu'est-ce qu'un LLM ?](/docs/foundations) <LevelBadge level="beginner" />
5. [Productivité au quotidien](/docs/playbooks/productivity) <LevelBadge level="beginner" />

## 💼 Le travailleur du savoir · ~2 h

Vous voulez que l'IA accélère votre vrai travail — rédaction, recherche, analyse.

1. La voie du novice curieux (ci-dessus), puis…
2. [Bibliothèque de modèles de prompts](/docs/prompting) <LevelBadge level="all" />
3. [Les hallucinations & comment les réduire](/docs/foundations) <LevelBadge level="intermediate" />
4. Votre playbook : [Rédaction](/docs/playbooks), [Recherche](/docs/playbooks), ou [Analyse de données](/docs/playbooks)
5. [Projects : des espaces de travail persistants](/docs/claude-app) et [Modèles de prompts réutilisables](/docs/templates)

## 🛠️ Le bâtisseur / développeur · ~½ journée

Vous écrivez du code et voulez personnaliser Claude Code et construire sur l'API.

1. [Ce qu'est Claude Code](/docs/claude-code/what-is-claude-code) <LevelBadge level="beginner" />
2. [CLAUDE.md & fichiers de mémoire](/docs/claude-code) <LevelBadge level="beginner" /> + [Mode Plan](/docs/claude-code)
3. [Permissions](/docs/claude-code) et [settings.json](/docs/claude-code) <LevelBadge level="intermediate" />
4. [Votre premier appel API](/docs/api/first-call) → [Tool Use](/docs/api) → [Construire des agents](/docs/api) <LevelBadge level="advanced" />
5. [Tutoriel : personnaliser Claude Code pour un vrai dépôt](/docs/walkthroughs) <LevelBadge level="intermediate" />
6. [Sécurité & usage responsable](/docs/security) <LevelBadge level="advanced" />

## 👥 Le responsable d'équipe · ~3 h

Vous déployez l'IA dans une équipe et vous souciez de cohérence, de sécurité et de coût.

1. Les essentiels de la voie du bâtisseur (1–3 ci-dessus)
2. [Plugins & places de marché](/docs/claude-code) — distribuez une boîte à outils partagée <LevelBadge level="advanced" />
3. [Compromis coût & latence](/docs/foundations) et [Choisir un modèle](/docs/api/choosing-a-model) <LevelBadge level="intermediate" />
4. [Sécuriser les agents](/docs/security) et [Renforcer les exécutions autonomes](/docs/security) <LevelBadge level="advanced" />
5. [Usage responsable & vérification](/docs/security) — définissez les normes de l'équipe <LevelBadge level="all" />
