Claude est **multimodal** : vous pouvez inclure des **images et des documents** (comme des PDF) dans un message et poser des questions à leur sujet — extraire des données, décrire un graphique, résumer un contrat, lire une capture d'écran.

## Trois façons d'envoyer un fichier

<Steps items={[{title: "Base64", body: "Encodez les octets en ligne dans la requête. Le plus simple pour les cas ponctuels."}, {title: "URL", body: "Pointez vers un fichier hébergé."}, {title: "Files API", body: "Téléversez une fois, puis référencez-le par file_id à travers de nombreuses requêtes (évite de re-téléverser de gros fichiers)."}]} />

Voici à quoi ressemble un bloc de contenu image aux côtés d'une question textuelle :

```python
# Conceptual: an image content block alongside text (see docs for exact fields)
messages=[{
  "role": "user",
  "content": [
    {"type": "image", "source": {"type": "base64", "media_type": "image/png", "data": b64}},
    {"type": "text", "text": "What does this chart show? Give the top 3 takeaways."},
  ],
}]
```

Les PDF fonctionnent de façon similaire (un bloc de contenu document) ; Claude peut lire le texte et la mise en page visuelle, et vous pouvez demander des **citations** renvoyant à l'endroit du document d'où provient une réponse.
