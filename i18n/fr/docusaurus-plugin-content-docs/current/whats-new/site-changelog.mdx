Modifications apportées à **AILmanac lui-même** — structure, fonctionnalités et ajouts majeurs de contenu. (Pour les changements concernant *Claude*, consultez [Ce mois-ci dans Claude](/docs/whats-new/this-month).) Le format suit [Keep a Changelog](https://keepachangelog.com/).

## Non publié

### Ajouté
- Sections complètes **Claude Code** et **API Claude** (toutes les pages rédigées).
- Sections **Fondamentaux de l'IA**, **Prompting**, **Claude.ai & Apps**, **Playbooks**, **Templates** et **Sécurité** terminées.
- Widgets interactifs : **Prompt Builder**, **Model Picker**, **Token Estimator**.
- **7 packs de skills** (`/skills`) pour différents types d'utilisateurs.
- Carte sociale **Open Graph** brandée ; thème premium + page d'accueil animée.
- Système de fraîcheur : `data/models.json` comme source de vérité + tampons `<VerifyNote>`.

### Modifié
- Refonte visuelle premium (typographie, palette, logo personnalisé).
