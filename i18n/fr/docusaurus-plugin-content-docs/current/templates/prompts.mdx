Des prompts prêts à l'emploi pour les tâches courantes. Ils utilisent des [balises XML](/docs/prompting/xml-tags) pour une structure propre et intègrent les habitudes des [Bases du prompting](/docs/prompting/basics). Remplissez les `{accolades}`.

## Résumer (ancré)

```text
<instructions>
Summarize the document for {audience} in {N} bullet points. Use ONLY the document;
if something isn't there, say "not stated." Then list any action items.
</instructions>
<document>
{paste}
</document>
```
*Attendu :* des puces concises + des actions à mener. *Échec courant :* il invente des faits → la clause « ONLY the document » s'en prémunit.

## Extraire des données structurées

```text
Extract the following from the text as JSON: {field1, field2, field3}.
Return ONLY valid JSON, no prose. If a field is missing, use null.
<text>{paste}</text>
```
*À associer avec* la [sortie structurée](/docs/api/structured-output) pour plus de fiabilité.

## Réécrire / éditer

```text
Rewrite the text to be {clearer/shorter/more formal} without changing meaning.
Then list the changes you made as bullets.
<text>{paste}</text>
```

## Revue de code

```text
Review this diff for correctness bugs, missing edge cases, and security issues.
Report ONLY high-confidence findings as a checklist with file:line. Diff:
<diff>{paste}</diff>
```

## Aide à la décision

```text
I'm deciding between {A} and {B} for {situation}. Build a trade-off table across
the criteria that matter, then recommend one and name the biggest risk.
```

## Méta-prompt (améliorer un prompt)

```text
Here's my prompt: "{prompt}". Ask me the 3 questions that would most improve it,
then rewrite it with sensible defaults.
```

## Wrapper rôle + garde-fou

```text
You are {role}. {task}.
Constraints: {format, length, tone}. If you're unsure or lack info, say so rather
than guessing.
```

:::tip Faites-les vôtres
Enregistrez ceux que vous utilisez comme [commandes slash](/docs/templates/slash-commands) ou comme [Projet](/docs/claude-app/projects) pour ne plus jamais les retaper.
:::
