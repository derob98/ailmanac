Au-delà du texte à l'écran, Claude peut produire de **vrais fichiers téléchargeables** — documents Word, présentations PowerPoint, feuilles de calcul Excel, PDF — mis en forme et prêts à l'emploi. Idéal lorsqu'il vous faut un livrable, pas seulement une réponse.

## Ce que vous pouvez demander

- **Word (.docx)** — rapports, lettres, propositions, documents mis en forme.
- **PowerPoint (.pptx)** — présentations à partir d'un plan ou de notes.
- **Excel (.xlsx)** — feuilles de calcul avec données, formules, modèles simples.
- **PDF** — versions prêtes à imprimer de ce qui précède.

## Comment obtenir un bon fichier

1. **Décrivez le livrable** : type, public, longueur, sections, ton.
2. **Fournissez le contenu ou la source** (notes, données, un plan) — ou demandez à Claude de le rédiger d'abord.
3. **Demandez le fichier** (« exporte ceci en .docx »), puis **itérez** (« ajoute une page de titre », « transforme le tableau en graphique », « resserre la diapositive 3 »).

```text
Turn these notes into a 6-slide investor update deck (.pptx): traction, revenue,
roadmap, team, ask. Clean and minimal. Notes: {paste}
```

## Conseils

- **Donnez une structure.** Des titres / sections dans votre demande → un résultat plus propre.
- **Fournissez de vraies données** pour les feuilles de calcul — et **vérifiez** tout chiffre calculé ([Hallucinations](/docs/foundations/hallucinations)) ; ne diffusez pas de chiffres que vous n'avez pas contrôlés.
- **Affinez avant d'exporter** — il est plus facile de corriger dans la conversation que dans le fichier final.
- Ces capacités reposent sur les [Skills](/docs/claude-code/skills) en coulisses — la même idée que vous pouvez créer vous-même.
