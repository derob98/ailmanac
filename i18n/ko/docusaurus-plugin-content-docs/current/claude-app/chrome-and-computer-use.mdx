브라우저/컴퓨터 사용의 가용성(베타 상태, 플랫폼, 요금제)과 정확한 동작은 자주 바뀝니다 — 최신 세부 사항과 한계는 공식 문서에서 확인하세요.
</VerifyNote>

Claude는 단순히 대화만 하는 것이 아니라 **소프트웨어를 조작**할 수 있습니다: 브라우저에서 웹 페이지를 읽고 작업을 수행하거나, 컴퓨터를 제어해(커서 이동, 클릭, 입력) 여러 앱에 걸친 작업을 완료할 수 있습니다. 강력하지만, 가드레일을 이해해 둘 가치가 있습니다.

## 서로 관련된 두 가지 기능

- **Claude in Chrome** — 현재 페이지를 읽고 여러분을 대신해 페이지 내에서 작업(탐색, 클릭, 양식 작성)을 수행할 수 있는 브라우저 측 에이전트입니다.
- **컴퓨터 사용** — 네이티브 앱과 여러 앱에 걸친 워크플로를 조작하기 위해 데스크톱을 더 일반적으로 제어합니다(스크린샷 + 마우스/키보드).

다른 방식으로는 번거로운 작업에 사용하세요: 여러 페이지에 걸친 정보 수집, 반복적인 양식 작성, 여러 단계의 웹 작업 등.

## 안전 모델 (먼저 읽어 보세요)

AI가 여러분을 대신해 클릭하고 입력하도록 허용하는 것은 정확히 [프롬프트 인젝션](/docs/security/prompt-injection)이 위험해지는 지점입니다: 악의적인 웹 페이지에는 에이전트를 탈취하려는 지시가 포함될 수 있습니다.

:::warning 링크와 페이지를 신뢰할 수 없는 것으로 취급하세요
- 중대한 작업에는 **곁에서 감독하세요** — 작업 도중에 자리를 비우지 마세요.
- 민감한 계정(은행, 관리자)을 **무심코 다루게 하지 마세요**.
- **목적지를 검증하세요** — 에이전트는 의심스러운 링크를 따라가거나 페이지에 숨겨진 지시에 따라 행동해서는 안 됩니다.
- **최소 권한** — 에이전트가 행동해서는 안 되는 곳에서는 보기만 하고 행동하지 못하도록 제한/권한 모드가 바로 그 목적으로 존재합니다.
:::

## 적합한 작업 vs 위험한 작업

| 적합한 작업 | 주의가 필요한 작업 |
|---|---|
| 여러 페이지에 걸친 읽기 전용 조사 | 돈을 쓰는 모든 작업 |
| 알려진, 무해한 양식 작성 | 민감한 계정에 로그인하기 |
| 여러분이 감독하는 정보 추출 | 모르는 사람의 이메일/페이지에 따라 행동하기 |
