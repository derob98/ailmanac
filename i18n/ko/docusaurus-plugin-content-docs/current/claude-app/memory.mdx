채팅 간 메모리의 동작과 가용성은 요금제에 따라 다르며 바뀝니다 — 현재 세부 사항은 앱/도움말 센터에서 확인하세요.
</VerifyNote>

기본적으로 각 대화는 [백지 상태](/docs/foundations/tokens-and-context)입니다. **메모리** 기능을 사용하면 Claude가 여러분에 대한 유용한 사실을 채팅 간에 이어 갈 수 있어 — 매번 자기소개를 다시 할 필요가 없습니다.

## 여기서 "메모리"가 의미하는 것

사람들이 "메모리"라고 부르는 별개의 것들이 몇 가지 있습니다 — 혼동하지 마세요:

| 종류 | 무엇인가 |
|---|---|
| **컨텍스트 윈도우** | 한 채팅 *안에서의* 작업 기억(채팅이 끝나면 잊힘) — [토큰 및 컨텍스트](/docs/foundations/tokens-and-context)를 참고하세요 |
| **채팅 간 메모리** | Claude가 여러분에 대한 사실을 대화 *전반에 걸쳐* 기억함(이 페이지의 주제) |
| **[사용자 지정 지침](/docs/claude-app/custom-instructions)** | *여러분이* 명시적으로 설정하는 고정된 사실 |
| **[Projects](/docs/claude-app/projects)** | 작업 공간 범위로 한정된 지식 |

## 채팅 간 메모리가 도움이 되는 방식

활성화되면 Claude는 여러분의 선호, 진행 중인 프로젝트, 공유한 세부 사항 같은 것들을 떠올릴 수 있어 — 매 세션마다 낯선 사람이 아니라 연속적인 어시스턴트처럼 느껴지게 합니다. 가용성과 제어 옵션은 요금제에 따라 다릅니다.

## 통제력을 유지하세요

- 기억된 내용을 **검토하고 편집하세요**(앱이 제어 기능을 제공하는 경우).
- 백지 상태나 더 높은 프라이버시를 원하면 메모리를 **지우거나 비활성화하세요**.
- 중요한 사실의 정확성을 **메모리에 의존하지 마세요** — 핵심 세부 사항은 떠올린 내용을 믿기보다 확인하세요.
- **프라이버시에 유의하세요** — 메모리는 정보를 지속적으로 보관합니다; 보관되기를 원치 않는 민감한 데이터는 저장하지 마세요. [개인정보 보호](/docs/foundations/privacy)를 참고하세요.

:::tip 명시적인 것이 암시적인 것을 이긴다
*반드시* 적용되어야 하는 사실(여러분의 역할, 단위, 확고한 규칙)이라면, [사용자 지정 지침](/docs/claude-app/custom-instructions)을 설정하거나 [Project](/docs/claude-app/projects)를 사용하세요 — 메모리가 떠올려 주기를 바라는 것보다 더 신뢰할 수 있습니다.
:::
