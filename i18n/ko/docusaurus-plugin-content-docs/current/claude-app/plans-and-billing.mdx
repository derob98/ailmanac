요금제 이름, 가격, 한도는 정기적으로 바뀝니다 — 현재 수치는 항상 공식 가격 페이지에서 확인하세요. 여기서는 의도적으로 금액을 명시하지 않습니다.
</VerifyNote>

어떤 Claude 요금제가 필요할까요? 다음 달이면 낡아 버릴 가격을 인용하지 않고 생각하는 방법을 소개합니다(가격은 [공식 가격 페이지](https://www.anthropic.com/pricing)에서 확인하세요).

## 목적별 티어

| 티어 | 대략적인 대상 |
|---|---|
| **Free** | Claude를 체험하거나 가벼운 개인 용도 |
| **Pro** | 일상적인 개인 용도 — 더 많은 사용량, 더 많은 기능 |
| **Max** | Pro 한도에 부딪히는 헤비 개인 사용자 |
| **Team** | 소규모 팀 — 공유 작업 공간과 관리 기능 |
| **Enterprise** | 관리, 보안, 규모, 통제가 필요한 조직 |

(정확한 이름/구성은 바뀔 수 있습니다 — 가격 페이지에서 확인하세요.)

## "한도"가 어떻게 느껴지는가

소비자 요금제는 **일정 기간 동안 사용할 수 있는 양**(시간이 지나면 초기화되는 메시지/사용량)을 제한하며, 상위 티어일수록 그 상한이 높아집니다. 한도에 도달하면 보통 초기화될 때까지 기다리거나 업그레이드하라는 안내를 받습니다 — 작업이 사라지는 것은 아니며, 잠시 멈추는 것뿐입니다.

더 무거운 모델(그리고 더 긴 대화)은 할당량을 더 빠르게 소모합니다. 한도에 자주 부딪힌다면, 새 주제마다 **새 채팅을 시작하고** 프롬프트를 집중적으로 유지하는 것이 빠른 해결책입니다([컨텍스트 관리](/docs/foundations/tokens-and-context)와 같은 개념입니다).

## 선택하는 방법

1. **Free로 시작하세요.** 실제로 무엇에 쓰는지 파악하세요.
2. 한도가 실제 작업을 방해할 때 **Pro로 업그레이드하세요**.
3. 여전히 상한에 부딪히는 파워 유저라면 **Max로 가세요**.
4. 동료가 공유 접근, 관리, 컴플라이언스를 필요로 할 때 **Team/Enterprise**를 선택하세요.

## 앱 vs API 청구 — 서로 다른 것

- **앱 요금제** = 사용 한도가 있는 정액 구독(이 페이지).
- **API 청구** = 여러분이 만드는 것에 대해 [토큰](/docs/api/tokens-and-pricing)당 지불. 다른 모델, 다른 페이지입니다.
