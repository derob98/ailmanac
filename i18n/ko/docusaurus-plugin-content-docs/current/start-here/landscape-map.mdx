AILmanac은 진공 속에 존재하지 않으며, 모든 것을 대체하는 척하지도 않습니다. 여기 최고의 자료들에 대한 정직한 지도가 있습니다 — 그리고 우리 **대신** 그것들을 언제 써야 하는지도요.

## 진실의 원천

- **[Anthropic 문서](https://docs.anthropic.com)** — Claude, API, 모델, 기능에 대한 권위 있는 참고서. **AILmanac과 문서가 서로 다를 경우, 문서가 이깁니다.** 정확하고 최신인 세부 사항은 문서를, 의견이 분명하고 단계별이며 실습으로 배우는 안내는 우리를 이용하세요.

## 공식 Anthropic 자료

- **[프롬프트 라이브러리](https://docs.anthropic.com/en/resources/prompt-library/library)** — 바로 쓸 수 있는 예시 프롬프트.
- **[Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)** — 실행 가능한 API 레시피(도구 사용, RAG, 비전).
- **[Claude Code 문서](https://docs.anthropic.com/en/docs/claude-code/overview)** — CLI/IDE 참고서.
- **[Anthropic Academy / Learn](https://www.anthropic.com/learn)** — 무료 가이드 강좌.
- **[Model Context Protocol](https://modelcontextprotocol.io)** — 개방형 MCP 표준 + 서버 디렉터리.

## 커뮤니티 & 일반

- **[awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code)** — 엄선된 명령어, 훅, 워크플로.
- **[Prompt Engineering Guide](https://www.promptingguide.ai)** — 깊이 있고 벤더 중립적인 프롬프트 참고서.
- **[Learn Prompting](https://learnprompting.org)** — 체계적이고 초보자 친화적인 프롬프트 강좌.

## 무엇을 언제 쓸까

| 원하는 것… | 갈 곳 |
|---|---|
| 정확한 최신 API/기능 세부 사항 | 공식 문서 |
| 복사·붙여넣기 예시 프롬프트 | 프롬프트 라이브러리 / 우리 [템플릿](/docs/templates/prompts) |
| 실행 가능한 API 코드 | Cookbook |
| 의견이 분명하고 수준이 태깅된 경로 | **AILmanac** |
| 프롬프트에 대한 깊은 이론 참고서 | promptingguide.ai |
| 가이드 강좌 | Anthropic Academy |

## AILmanac의 위치

우리는 **의견이 분명한 현장 가이드**입니다: 하나의 권장 방법을 먼저, 수준으로 태깅하고, 템플릿이 풍부하며, 최신성에 정직하고, Claude **와** 일반 AI 개념을 모두 아우릅니다. 우리를 적절한 시점에 적절한 심화 자료로 안내하는 지도라고 생각하세요 — 그리고 그 모든 것이 이해되도록 기본기를 가르쳐 주는 곳이라고요.
