프로그래머가 되지 않고도 AI를 잘 쓰고 싶은 분.

1. [첫 5분](/docs/start-here/your-first-5-minutes) <LevelBadge level="beginner" />
2. [어떤 Claude를 써야 할까?](/docs/start-here/which-claude) <LevelBadge level="beginner" />
3. [프롬프트 기초](/docs/prompting/basics) <LevelBadge level="beginner" />
4. [LLM이란 무엇인가?](/docs/foundations) <LevelBadge level="beginner" />
5. [일상 생산성](/docs/playbooks/productivity) <LevelBadge level="beginner" />

## 💼 지식 노동자 · 약 2시간

실제 업무 — 글쓰기, 리서치, 분석 — 를 AI로 더 빠르게 하고 싶은 분.

1. 호기심 많은 입문자 트랙(위), 그다음…
2. [프롬프트 패턴 라이브러리](/docs/prompting) <LevelBadge level="all" />
3. [환각과 그것을 줄이는 법](/docs/foundations) <LevelBadge level="intermediate" />
4. 당신의 플레이북: [글쓰기](/docs/playbooks), [리서치](/docs/playbooks), 또는 [데이터 분석](/docs/playbooks)
5. [프로젝트: 지속되는 작업 공간](/docs/claude-app)과 [재사용 가능한 프롬프트 템플릿](/docs/templates)

코드를 작성하며 Claude Code를 맞춤 설정하고 API 위에서 무언가를 만들고 싶은 분.

1. [Claude Code란 무엇인가](/docs/claude-code/what-is-claude-code) <LevelBadge level="beginner" />
2. [CLAUDE.md & 메모리 파일](/docs/claude-code) <LevelBadge level="beginner" /> + [플랜 모드](/docs/claude-code)
3. [권한](/docs/claude-code)과 [settings.json](/docs/claude-code) <LevelBadge level="intermediate" />
4. [첫 API 호출](/docs/api/first-call) → [도구 사용](/docs/api) → [에이전트 구축](/docs/api) <LevelBadge level="advanced" />
5. [실습: 실제 저장소에 맞춰 Claude Code 맞춤 설정하기](/docs/walkthroughs) <LevelBadge level="intermediate" />
6. [보안 & 책임 있는 사용](/docs/security) <LevelBadge level="advanced" />

## 👥 팀 리드 · 약 3시간

팀에 AI를 도입하고 있으며 일관성, 안전성, 비용을 중요하게 여기는 분.

1. 빌더 트랙의 핵심(위 1–3번)
2. [플러그인 & 마켓플레이스](/docs/claude-code) — 공유 툴킷 배포하기 <LevelBadge level="advanced" />
3. [비용 & 지연 시간 트레이드오프](/docs/foundations)와 [모델 선택하기](/docs/api/choosing-a-model) <LevelBadge level="intermediate" />
4. [에이전트 보안](/docs/security)과 [자율 실행 강화하기](/docs/security) <LevelBadge level="advanced" />
5. [책임 있는 사용 & 검증](/docs/security) — 팀 규범 정하기 <LevelBadge level="all" />
