**AILmanac**은 Claude는 물론 어떤 AI든 최대한 활용하기 위한, 무료이며 커뮤니티가 만든 **연감(almanac)**입니다. 연감이란 다시 찾아오게 되는 참고서를 말합니다. 완결적이고, 알기 쉬우며, 늘 최신으로 유지됩니다. 바로 그것이 이곳의 목표입니다.

## 누구를 위한 것인가

의도적으로, 모두를 위한 것입니다. 그리고 모든 페이지의 수준을 알려드리므로 길을 잃거나 무시당하는 느낌을 받을 일이 없습니다:

- <LevelBadge level="beginner" /> AI를 거의 써본 적이 없고 확실한 첫 성공을 원하는 분.
- <LevelBadge level="intermediate" /> 이미 Claude를 사용하고 있으며 눈에 띄게 더 나은 결과를 원하는 분.
- <LevelBadge level="advanced" /> Claude Code, API, 에이전트, MCP로 무언가를 만드는 분.

## 무엇이 다른가

이미 훌륭한 자료들이 있습니다([더 랜드스케이프 맵](/docs/start-here/landscape-map) 참고). AILmanac은 링크 모음이 **아니며** 공식 문서의 복사본도 **아닙니다**. 이곳은:

- **의견이 분명합니다** — *하나의 권장 방법*을 먼저 알려드리고, 그다음 대안을 제시합니다.
- **수준이 태깅되어 있습니다** — 모든 페이지에 배지가 붙어 있어 자신에게 맞는 내용을 읽을 수 있습니다.
- **템플릿이 풍부합니다** — 30초 만에 바로 쓸 수 있는 [복사·붙여넣기 스타터](/docs/templates).
- **최신성에 정직합니다** — 변하는 사실(모델 이름, 가격, 기능)에는 "마지막 확인" 날짜와 출처 링크가 붙습니다.

## 이것이 아닌 것

:::warning 공식 문서가 진실의 원천입니다
AILmanac과 [공식 Anthropic 문서](https://docs.anthropic.com)가 서로 다를 경우, **공식 문서가 이깁니다**. AI 도구는 빠르게 바뀝니다. 우리는 최신을 유지하려 노력하지만, 중요한 내용은 항상 우리가 링크한 출처에서 확인하세요.
:::

또한 우리는 당신의 데이터를 저장하지 않고, 계정을 볼 수 없으며, 중대한 작업에서 당신을 대신해 결정을 내리지 않습니다 — [책임 있는 사용](/docs/security)을 참고하세요.
