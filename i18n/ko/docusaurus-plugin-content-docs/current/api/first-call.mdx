API는 *당신의* 프로그램이 Claude와 대화할 수 있게 해줍니다. 세 단계입니다: 키를 받고, SDK를 설치하고, 메시지를 보냅니다.

## 1. API 키 받기

[Anthropic Console](https://console.anthropic.com)에서 키를 하나 생성하세요. 그런 다음 코드에 절대 남지 않도록 환경 변수로 설정하세요:

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

:::warning 키를 절대 커밋하지 마세요
키는 환경 변수나 시크릿 매니저에 보관하세요 — 소스 관리에는 절대 넣지 마세요. [보안](/docs/security) 참고.
:::

## 3. 호출하기

모든 요청은 `messages`의 리스트입니다. 모델은 `content`로 응답합니다.

<Tabs groupId="lang">
<TabItem value="python" label="Python">

```python
import anthropic

client = anthropic.Anthropic()  # reads ANTHROPIC_API_KEY from the environment

message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "In one sentence, what is an API?"}
    ],
)

print(message.content[0].text)
```

</TabItem>
<TabItem value="ts" label="TypeScript">

```ts
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic(); // reads ANTHROPIC_API_KEY from the environment

const message = await client.messages.create({
  model: "claude-sonnet-4-6",
  max_tokens: 1024,
  messages: [
    { role: "user", content: "In one sentence, what is an API?" },
  ],
});

console.log(message.content[0].text);
```

</TabItem>
<TabItem value="curl" label="cURL">

```bash
curl https://api.anthropic.com/v1/messages \
  --header "x-api-key: $ANTHROPIC_API_KEY" \
  --header "anthropic-version: 2023-06-01" \
  --header "content-type: application/json" \
  --data '{
    "model": "claude-sonnet-4-6",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "In one sentence, what is an API?"}
    ]
  }'
```

</TabItem>
</Tabs>

## 방금 무슨 일이 일어났나

- **`model`** — 어떤 Claude를 사용할지. 무턱대고 하드코딩하지 마세요; [모델 선택](/docs/api/choosing-a-model) 참고.
- **`max_tokens`** — 응답 길이(단위: [토큰](/docs/foundations))의 상한입니다. 컨텍스트 윈도우를 설정하는 것이 *아닙니다*.
- **`messages`** — 지금까지의 대화. API는 **무상태(stateless)**입니다: 대화를 이어가려면 매번 전체 히스토리를 다시 보내야 합니다.
- **`system`** (선택) — 이 호출에서 Claude의 역할을 설정하는 최상위 지시문.
