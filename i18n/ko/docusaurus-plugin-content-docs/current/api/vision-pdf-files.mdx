Claude는 **멀티모달**입니다: 메시지에 **이미지와 문서**(예: PDF)를 포함해 질문할 수 있습니다 — 데이터 추출, 차트 설명, 계약서 요약, 스크린샷 읽기.

## 파일을 보내는 세 가지 방법

<Steps items={[{title: "Base64", body: "바이트를 요청에 인라인으로 인코딩합니다. 일회성에 가장 간단합니다."}, {title: "URL", body: "호스팅된 파일을 가리킵니다."}, {title: "Files API", body: "한 번 업로드하고, 여러 요청에서 file_id로 참조합니다(큰 파일 재업로드를 피합니다)."}]} />

이미지 콘텐츠 블록이 텍스트 질문과 함께 어떻게 보이는지:

```python
# Conceptual: an image content block alongside text (see docs for exact fields)
messages=[{
  "role": "user",
  "content": [
    {"type": "image", "source": {"type": "base64", "media_type": "image/png", "data": b64}},
    {"type": "text", "text": "What does this chart show? Give the top 3 takeaways."},
  ],
}]
```

PDF도 비슷하게 작동합니다(문서 콘텐츠 블록); Claude는 텍스트와 시각적 레이아웃을 읽을 수 있고, 답변이 문서의 어디에서 왔는지로 되돌아가는 **인용**을 요청할 수 있습니다.
