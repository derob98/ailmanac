**AILmanac 자체**의 변경 사항 — 구조, 기능, 주요 콘텐츠 추가. (*Claude*의 변경 사항은 [이달의 Claude](/docs/whats-new/this-month)를 참고하세요.) 형식은 [Keep a Changelog](https://keepachangelog.com/)를 따릅니다.

## 미공개

### 추가됨
- 전체 **Claude Code** 및 **Claude API** 섹션(모든 페이지 작성 완료).
- **AI 기초**, **프롬프팅**, **Claude.ai 및 앱**, **플레이북**, **템플릿**, **보안** 섹션 완성.
- 인터랙티브 위젯: **Prompt Builder**, **Model Picker**, **Token Estimator**.
- 다양한 사용자 유형을 위한 **7개의 스킬 팩**(`/skills`).
- 브랜드 **Open Graph** 소셜 카드, 프리미엄 테마 + 애니메이션 홈페이지.
- 최신성 시스템: 단일 진실 공급원인 `data/models.json` + `<VerifyNote>` 스탬프.

### 변경됨
- 프리미엄 비주얼 리디자인(타이포그래피, 팔레트, 커스텀 로고).

## 참고

- 이 프로젝트는 커뮤니티 프로젝트입니다 — 기여자 목록은 저장소에서 확인하세요.
- 여기에 변경 사항을 기록하고 싶으신가요? PR에서 언급해 주세요([기여하기](/docs/contribute/contribute-in-10-minutes)).
- 변경 사항을 구독하려면 [GitHub 저장소](https://github.com/derob98/ailmanac)를 워치하세요.
