# 프런티어

AILmanac의 대부분은 기능, 설정, 가격, 명령어 같은 것들이 어떻게 작동하는지를 다룹니다. **프런티어**는 다릅니다. 여기에는 **사고의 도구** — 어떤 버튼을 누르는지가 아니라, 당신이 결정을 내리는 방식 자체를 바꾸는 렌즈들이 담겨 있습니다.

이곳에는 두 종류의 콘텐츠가 있습니다:

1. **AILmanac만의 멘탈 모델.** 우리가 정말로 유용하다고 느꼈고, 다른 곳에서 정리된 형태로 본 적이 없는 독창적인 틀입니다. 우리는 이것들이 외부에서 확립된 사실이 아니라 *우리만의* 주관적인 렌즈라는 점을 솔직하게 밝힙니다. 그중 하나가 당신에게 무언가를 마침내 이해시켜 준다면, 그게 바로 핵심입니다.
2. **실재하지만 활용되지 못한 아이디어.** 사실이고 중요하지만 대부분의 가이드가 건너뛰는 개념들입니다 — 보통은 하나의 기능이나 스크린샷에 묶여 있지 않기 때문입니다.

좋은 멘탈 모델은 두 가지 방식으로 제 값을 합니다. 현실과 부딪혀도 살아남고, 다른 상황으로 전이됩니다. 여기 실린 틀들은 의도적으로 특정 모델 버전이나 가격에 관한 것이 **아닙니다**. 그런 것들은 매주 바뀝니다(그것이 바로 첫 페이지의 주제입니다). 이 렌즈들은 그것들보다 오래 살아남도록 만들어졌습니다.

## 이것들을 읽는 법

각 페이지를 교리가 아니라 도구로 대하세요. 유용한 부분을 취하고 나머지는 두세요. 점심때쯤이면 잊어버릴 열 가지 팁보다, 당신의 사고를 재정렬해 주는 하나의 틀을 드리는 편이 낫습니다.

이것들 중 어느 것의 더 날카로운 버전을 발견했거나 — 공유할 만한 당신만의 렌즈가 있다면 — 그것이야말로 우리가 바라는 것입니다. [10분 만에 기여하기](/docs/contribute/contribute-in-10-minutes)를 참고하세요.
