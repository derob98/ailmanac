데이터 처리 및 학습 정책은 공급자, 제품, 요금제(소비자용 vs API vs 엔터프라이즈)에 따라 다르고 시간이 지나면서 바뀝니다. 항상 현재 약관을 공급자에게 확인하세요.
</VerifyNote>

진짜로 중요한 단순한 질문: **당신이 AI에 보내는 텍스트에 무슨 일이 일어나는가?** 답은 공급자, 제품, 당신의 요금제에 따라 다릅니다 — 그래서 안전한 습관은 *붙여 넣기 전에 아는 것*입니다.

## 당신의 도구에 대해 답해야 할 질문들

- **내 입력이 모델 학습에 사용되는가?** 이것은 크게 다릅니다 — 소비자용 챗, API, 엔터프라이즈 등급이 흔히 차이가 납니다. 많은 **API/엔터프라이즈** 상품은 기본적으로 당신의 데이터로 학습하지 **않습니다**. 일부 소비자용 설정은 당신의 선택에 따라 학습할 수 있습니다. 확인하세요.
- **데이터는 얼마나 오래 보관되며**, 누가 접근할 수 있는가?
- **어디에서 처리되며**(지역/소재지), 암호화되어 있는가?
- 민감한 작업을 위한 **무보존(zero-retention) 또는 비즈니스/엔터프라이즈 옵션**이 있는가?

## 실용적인 경험칙

- **비밀 정보를 붙여 넣지 마세요** — API 키, 비밀번호, 토큰 — 어떤 AI 도구에도. 절대로.
- **개인정보를 최소화하세요.** 작업에 필요한 것만 보내고, 이름·ID·계좌번호는 가능할 때 가리세요.
- **다른 사람의 데이터는 그들의 데이터입니다.** 고객 PII, 건강, 재무 정보를 다룰 때 주의하세요 — 법적 의무(GDPR, HIPAA 등)가 있을 수 있습니다.
- **올바른 등급을 사용하세요.** 규제 대상이거나 기밀인 작업에는 개인 계정이 아니라 비즈니스/엔터프라이즈/무보존 옵션을 사용하세요.
- 데이터가 정말로 당신의 환경을 벗어나서는 안 될 때 **로컬/셀프호스팅 모델을 고려하세요** — 일부 성능을 완전한 통제와 맞바꾸는 것입니다.

## 온디바이스 vs 클라우드

| | 클라우드 (호스팅형) | 로컬 / 셀프호스팅 |
|---|---|---|
| 성능 | 최고 | 낮음 (하지만 개선 중) |
| 데이터가 내 기기를 벗어남 | 예 (공급자 약관에 따라) | 아니오 |
| 설정/비용 | 쉬움, 사용한 만큼 지불 | 더 많은 수고, 당신의 하드웨어 |
