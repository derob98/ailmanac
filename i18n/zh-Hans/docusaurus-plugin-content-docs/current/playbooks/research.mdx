AI 是一个极佳的调研*加速器*——而如果你盲目信任它,它也会很危险。用得好,它能把数小时的阅读压缩到几分钟;用得草率,它会信心满满地编造信息源。下面是讲纪律的那个版本。

## 高价值的调研动作

- 把一份长文档/论文**总结**成要点 + 影响。
- 把多个信息源**比较**成一张表(它们在哪些点上一致/分歧)。
- **构建全景图** —— 梳理某个主题上的参与方、各种方法或各方论点。
- 从文档中**提取**特定的事实/数字(附带引用出处)。
- **压力测试** ——“对这个观点最有力的反驳是什么?”

## 锚定纪律(不可妥协)

```text
Using ONLY the text I provide, summarize the key findings and quote the exact
sentence supporting each one. If something isn't in the text, say "not stated."
Text: """{paste}"""
```

- **把信息源喂给它。** 对任何要紧的内容,都不要让它凭记忆回想事实——粘贴原始材料,或使用能检索材料的工具([RAG](/docs/foundations/rag))。
- **要求提供引用。** “引用支持每条主张的那句话。”无依据的主张就会暴露出来。
- **把事实和推断区分开。** 让它标注出哪些是文中所述,哪些是它的解读。

:::warning 伪造引用是经典的陷阱
模型会编造出看起来很真实的论文、作者和网址。在你依赖某条引用之前,**核实它确实存在**。参见[幻觉](/docs/foundations/hallucinations)。
:::

## 一套调研流程

1. 自己(或借助检索工具)收集信息源。
2. 对每个信息源做带锚定 + 引用的总结。
3. 跨信息源综合成一份按主题归类的比较。
4. 找出空白/分歧以及“那又怎样(so what)”。
5. 把关键事实与原文逐一抽查核对。
