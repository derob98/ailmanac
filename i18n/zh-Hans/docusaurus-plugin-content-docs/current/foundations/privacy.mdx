一个简单却利害攸关的问题：**你发给 AI 的文本会发生什么？** 答案取决于提供商、产品和你的套餐——所以安全的习惯是*在粘贴之前就搞清楚*。

## 为你的工具回答的几个问题

- **我的输入会被用来训练模型吗？** 这差别很大——消费者版聊天、API 和企业版分级往往各不相同。许多 **API/企业版** 产品默认**不会**用你的数据训练；某些消费者版设置则可能会，取决于你的选择。要核实。
- **数据保留多久**，以及谁能访问它？
- **它在哪里处理**（区域/驻留），是否加密？
- **是否有零保留或商业/企业版选项**用于敏感工作？

## 实用经验法则

- **不要粘贴机密**——API 密钥、密码、令牌——绝不要粘进任何 AI 工具。
- **尽量减少个人数据。** 只发送任务所需的内容；尽可能脱敏姓名、ID、账号。
- **别人的数据是别人的。** 对客户 PII、健康或财务信息要谨慎——可能存在法律义务（GDPR、HIPAA 等）。
- **用对分级。** 对受监管或机密的工作，使用商业/企业/零保留选项，而非个人账户。
- **考虑本地/自托管模型**——当数据确实不能离开你的环境时，你用一些能力换来完全的掌控。

## 端侧 vs 云端

| | 云端（托管） | 本地 / 自托管 |
|---|---|---|
| 能力 | 最高 | 较低（但在改进） |
| 数据离开你的机器 | 是（按提供商条款） | 否 |
| 搭建/成本 | 简单、按用量付费 | 更费力、用你自己的硬件 |
