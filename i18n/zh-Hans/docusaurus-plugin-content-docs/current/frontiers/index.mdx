# 前沿

AILmanac 的大部分内容记录的是事物如何运作：功能、设置、价格、命令。**前沿**则不同。这里放的是**思维工具**——那些改变你做决策方式的视角，而不只是你该按哪个按钮。

这里有两类内容：

1. **AILmanac 自己的思维模型。** 我们觉得确实有用、且没在别处见过成体系打包的原创框架。我们坦率承认这些是*我们*带有立场的视角，而非外部既定事实。如果某个模型让你某件事终于豁然开朗，那就达到目的了。
2. **真实却被低估的观念。** 那些真实而重要、却被大多数指南跳过的概念——通常是因为它们没有绑定到某个具体功能或截图上。

一个好的思维模型靠两点来证明自己的价值：它经得起现实世界的检验，并且可以迁移。这里的框架刻意**不**针对任何单一的模型版本或价格。那些东西每周都在变（这恰恰是第一页的主题）。这些视角则是为了比它们活得更久。

## 该怎么读这些内容

把每一页当作工具，而不是教条。取走有用的部分，剩下的留着。我们宁愿给你一个能重组你思考方式的视角，也不愿给你十条到午饭时就忘光的小贴士。

如果你发现了其中任何一条的更精炼版本——或者你自己有值得分享的视角——那正是我们想要的。参见[十分钟参与贡献](/docs/contribute/contribute-in-10-minutes)。
