## 如何阅读一个条目

每条更新都按 **领域** 标记——🗣️ 应用（聊天/语音/移动端）、🛠️ Claude Code、🔌 API、🧠 模型——并回答 **“它对你意味着什么”**。

## 近期更新

> _种子条目——随着真实、有来源的更新发布，请替换为它们。_

- 🧠 **模型**——查看 [当前模型与定价](/docs/whats-new/models-and-pricing) 表格，了解最新的 Claude 模型阵容和 ID。*这意味着什么：* 在当前默认模型上开始新的构建，并在那里查找确切的 ID。
- 🛠️ **Claude Code**——功能更新频繁（技能、钩子、MCP、调度）。*这意味着什么：* 如果某个页面的 `last verified` 日期看起来已经过时，请对照 [官方更新日志](https://code.claude.com/docs/en/overview) 核实。
- 🔌 **API**——诸如结构化输出、扩展思考和提示缓存等能力在不断演进。*这意味着什么：* 在依赖确切参数之前，请重新查阅 [API 章节](/docs/api/first-call)。
