## AILmanac 的定位

我们是那本 **有主张的实地手册**：先给唯一推荐的做法、按难度打标签、模板丰富、对时效性诚实，并且横跨 Claude **和** 通用 AI 概念。把我们想成一张地图，在恰当的时机为你指向恰当的深度资源——并把基础讲清楚，让这一切都说得通。
