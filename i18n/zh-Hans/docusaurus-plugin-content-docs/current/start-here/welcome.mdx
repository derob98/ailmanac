**AILmanac** 是一份免费的、由社区共建的 **指南手册，帮助你充分发挥 Claude——以及任何 AI——的价值**。所谓"年鉴/手册"（almanac），是一份你会反复回来查阅的参考资料：内容完整、表述平实，并且持续保持最新。这正是本项目的目标。

## 这是为谁准备的

刻意面向所有人——而且我们会标明每一页的难度等级，让你既不会迷失方向，也不会被居高临下地对待：

- <LevelBadge level="beginner" /> 你几乎没用过 AI，想要一次稳稳的开门红。
- <LevelBadge level="intermediate" /> 你已经在用 Claude，希望结果明显更好。
- <LevelBadge level="advanced" /> 你用 Claude Code、API、智能体和 MCP 进行开发。

## 它有何不同

已经有很多优秀的资源（参见 [全景地图](/docs/start-here/landscape-map)）。AILmanac **不是** 一堆链接的堆砌，**也不是** 对官方文档的照搬。它是：

- **有主张的**——我们先告诉你 *唯一推荐的做法*，再给出其他备选方案。
- **标注难度的**——每一页都打了标签，让你只读适合自己的内容。
- **模板丰富的**——[即拷即用的起步模板](/docs/templates)，30 秒就能用上。
- **对时效性诚实的**——会变化的事实（模型名称、价格、功能）都附有"上次核验"日期和指向来源的链接。

## 它 *不是* 什么

:::warning 官方文档才是权威依据
当 AILmanac 与 [Anthropic 官方文档](https://docs.anthropic.com) 发生冲突时，**以官方文档为准**。AI 工具变化很快；我们尽力保持最新，但对任何关键内容，请务必对照我们提供的来源链接进行确认。
:::

我们也不会存储你的数据、看不到你的账户，也不会在高风险工作上替你做决定——参见 [负责任地使用](/docs/security)。
