## `.mcp.json` —— 声明服务器（项目共享）

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": { "GITHUB_TOKEN": "${GITHUB_TOKEN}" }
    },
    "postgres": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres", "${DATABASE_URL}"]
    }
  }
}
```

:::warning 不要把密钥放进文件
引用环境变量（`${GITHUB_TOKEN}`）—— 不要把令牌硬编码到一个会被提交的文件里。
:::

## 最小化 stdio 服务器（TypeScript）

一个暴露单个工具的微型服务器。请将处理函数调整为适配你的数据。

```ts
import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";
import { z } from "zod";

const server = new McpServer({ name: "echo", version: "1.0.0" });

server.tool(
  "echo",
  { text: z.string().describe("Text to echo back") },
  async ({ text }) => ({ content: [{ type: "text", text: `You said: ${text}` }] }),
);

await server.connect(new StdioServerTransport());
```

## 最小化 stdio 服务器（Python）

```python
from mcp.server.fastmcp import FastMCP

mcp = FastMCP("echo")

@mcp.tool()
def echo(text: str) -> str:
    """Echo the text back."""
    return f"You said: {text}"

if __name__ == "__main__":
    mcp.run()  # stdio transport
```

## 上线服务器之前

- **最小权限** —— 只给它所需的数据/操作权限（[保护智能体安全](/docs/security/securing-agents)）。
- **校验输入**；将错误作为结果返回，不要让程序崩溃。
- **审查第三方服务器**后再连接（[审查第三方代码](/docs/security/reviewing-third-party-code)）。
