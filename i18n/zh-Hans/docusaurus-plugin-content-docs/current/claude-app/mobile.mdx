Claude 不只是桌面上的一个标签页。**iOS 和 Android 应用**把它装进了你的口袋——非常适合捕捉灵感、快速提问，以及在路上使用[语音](/docs/claude-app/voice-mode)。

## 移动端擅长什么

- **随时随地捕捉。**电梯里的一个念头、一张白板的照片、一句快速的"总结一下这个"——对于那些原本会丢失的输入，移动端大放异彩。
- **语音优先。**在手机上说话往往比打字更快——参见[语音模式](/docs/claude-app/voice-mode)。
- **照片与截图。**拍下一份菜单、一张表单、一条报错信息、一个路牌，然后针对它提问（多模态输入）。
- **连续性。**对话和[项目](/docs/claude-app/projects)会同步，因此你可以在移动端开始、在桌面端完成。

## 桌面端更胜一筹的地方

- 长篇**写作和编辑**。
- 处理**大型文档**或大量文件。
- **[制品（Artifacts）](/docs/claude-app/artifacts)**以及任何受益于屏幕空间的场景。
- **开发**工作（[Claude Code](/docs/claude-code/what-is-claude-code) 是桌面/命令行工具）。

## 一个不错的移动端工作流

**用语音捕捉 → 在屏幕上打磨。**走路时用[语音](/docs/claude-app/voice-mode)把一个杂乱的想法一股脑说出来，之后再请 Claude *"把那段转写整理成一份清晰的大纲"*。手机负责收集原始素材；你随处都能润色。

:::tip 善用分享菜单
在移动端，你通常可以从其他应用把文本/链接/图片直接分享进 Claude——这是一种无需复制粘贴就能让它"总结一下这个"的快捷方式。
:::
