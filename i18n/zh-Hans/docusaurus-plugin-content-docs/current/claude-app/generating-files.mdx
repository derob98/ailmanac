除了屏幕上的文本，Claude 还能生成**真实、可下载的文件**——Word 文档、PowerPoint 演示文稿、Excel 电子表格、PDF——格式齐全、即用即得。当你需要的是一份交付成果而不仅仅是一个答案时，这非常合适。

## 你可以要求生成什么

- **Word（.docx）**——报告、信函、提案、带格式的文档。
- **PowerPoint（.pptx）**——从大纲或笔记生成幻灯片演示文稿。
- **Excel（.xlsx）**——含数据、公式和简单模型的电子表格。
- **PDF**——上述内容的可打印版本。

## 如何获得一份好文件

1. **描述交付成果**：类型、受众、长度、章节、语气。
2. **提供内容或来源**（笔记、数据、大纲）——或先请 Claude 起草。
3. **要求生成文件**（"把这个导出为 .docx"），然后**迭代**（"加一个标题页"、"把表格做成图表"、"精简第 3 页"）。

```text
Turn these notes into a 6-slide investor update deck (.pptx): traction, revenue,
roadmap, team, ask. Clean and minimal. Notes: {paste}
```

## 小贴士

- **提供结构。**请求中带上标题/章节 → 输出更整洁。
- 为电子表格**提供真实数据**——并**核实**任何计算出的数字（[幻觉](/docs/foundations/hallucinations)）；不要交付你没有核对过的数据。
- **在导出前打磨**——在聊天中修改比在最终文件里修改更容易。
- 这些能力在底层由 [Skills](/docs/claude-code/skills) 驱动——也就是你自己可以编写的那套理念。
