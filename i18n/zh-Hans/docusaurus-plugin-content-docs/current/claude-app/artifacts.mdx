**制品（Artifacts）**是 Claude 在聊天旁边的**侧边面板**中渲染出来的输出——一份文档、一张图表、一个可运行的应用、一张图示——你可以查看、使用并对其进行迭代，它独立于对话文本之外。

## 你可以创建什么

- **微型 Web 应用与工具**——一个计算器、一个测验、一个表单、一个小型交互演示。
- **文档**——你可以打磨并导出的结构化文稿。
- **可视化内容**——图表、图示和简单的数据仪表盘。
- 你可以阅读并运行的**代码**。

## 为什么它对非开发者如此强大

你可以通过描述来构建*真正可用*的东西——"给我做一个聚餐用的小费计算器"、"用这个 CSV 做一个仪表盘"——然后通过对话来打磨它（"加一个服务费字段"、"把按钮做大一点"）。这是**无需自己写代码就能用 AI 进行构建**最清晰的例子。

## 如何使用制品

1. **提出你想要的东西**，并给出具体信息（用途、输入、外观）。
2. **用平实的语言迭代**——Claude 会更新同一个制品。
3. 在面板中**使用它**；在支持的地方**导出/分享**。

```mermaid
flowchart LR
  A[Describe what you want] --> B[Claude builds an artifact]
  B --> C[You try it in the side panel]
  C --> D[“Tweak this…”] --> B
```

## 小贴士

- 对输入/输出和受众要**具体明确**——和好的[提示词](/docs/prompting/basics)一样。
- **小步迭代。**一次只改一处更容易做对。
- 对于重要用途，**核实制品计算出的任何逻辑/数字**（[幻觉](/docs/foundations/hallucinations)）。
