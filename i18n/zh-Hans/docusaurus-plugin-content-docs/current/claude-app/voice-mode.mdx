有时候，最快的输入方式就是你的声音。语音模式让你能**对 Claude 说话**并听取或阅读它的回复——非常适合捕捉灵感、出声思考，或免手动地使用 Claude。

## 它擅长什么

- **在路上捕捉灵感**——把一个杂乱的想法说出来；请 Claude 把它结构化。
- **免手动的时刻**——走路、通勤、做饭。
- **练习与头脑风暴**——出声地来回交流感觉很自然。
- **无障碍**——对许多人来说，说话可能比打字更轻松（参见[无障碍](/docs/playbooks/accessibility)）。

## 开始使用

1. 打开 Claude 应用（通常是移动应用；可用性各异——参见上方提示）。
2. 在聊天界面发起一次语音对话。
3. 自然地说话；说完时停顿。Claude 会作出回应，并把一份**转写**保存到对话中，方便你回顾或复制。
4. 你通常可以在**按住说话**和**免手动**两种方式之间切换，而且它支持**多语言**——用你思考时所用的语言说话。

## 获得良好语音效果的小贴士

- **把诉求放在前面。**"帮我列一个婚礼致辞的大纲——三分钟，温暖又有趣"比东拉西扯效果更好。
- **说出你想要的格式。**"给我五个要点"——否则口语回复可能会啰嗦。
- **用它来起草，用屏幕来打磨。**用语音捕捉，然后在同一对话里整理文字。

:::tip 语音输入，文本输出
一个强大的工作流：用语音一股脑说出想法，然后输入*"把那个整理成一封清晰的邮件"*——转写就成了你的原始素材。
:::
