De qual plano do Claude você precisa? Veja como pensar sobre isso — sem citar preços que estariam desatualizados no mês que vem (consulte a [página oficial de preços](https://www.anthropic.com/pricing) para esses).

## Os níveis, por propósito

| Nível | Mais ou menos para |
|---|---|
| **Free** | Experimentar o Claude, uso pessoal leve |
| **Pro** | Uso individual regular — mais uso, mais capacidade |
| **Max** | Usuários individuais intensivos que atingem os limites do Pro |
| **Team** | Pequenas equipes — espaço de trabalho compartilhado e gerenciamento |
| **Enterprise** | Organizações que precisam de administração, segurança, escala e controles |

(Os nomes/estrutura exatos podem mudar — confirme na página de preços.)

## Como os "limites" se manifestam

Os planos de consumo limitam **quanto você pode usar em uma janela** (mensagens/uso que se renovam ao longo do tempo), e os níveis mais altos elevam esses limites. Ao atingir um limite, normalmente você é solicitado a esperar até que ele se renove ou fazer upgrade — seu trabalho não é perdido, você apenas faz uma pausa.

Modelos mais pesados (e conversas mais longas) consomem sua cota mais rapidamente. Se você está atingindo os limites, uma solução rápida é **começar conversas novas** para novos tópicos e manter os prompts focados (mesma ideia do [gerenciamento de contexto](/docs/foundations/tokens-and-context)).

## Como escolher

1. **Comece com o Free.** Aprenda para que você realmente o usa.
2. **Faça upgrade para o Pro** quando os limites interromperem o trabalho de verdade.
3. **Vá para o Max** se você é um usuário avançado e ainda atinge os limites.
4. **Team/Enterprise** quando colegas precisarem de acesso compartilhado, administração ou conformidade.

## Cobrança de apps vs. API — coisas diferentes

- **Planos de app** = uma assinatura fixa com limites de uso (esta página).
- **Cobrança da API** = pague por [token](/docs/api/tokens-and-pricing) pelo que você constrói. Modelo diferente, página diferente.
