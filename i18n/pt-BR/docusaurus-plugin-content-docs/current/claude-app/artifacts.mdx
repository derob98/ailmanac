Os **artefatos** são saídas que o Claude renderiza em um **painel lateral** ao lado do chat — um documento, um gráfico, um app funcional, um diagrama — que você pode ver, usar e iterar, separados do texto da conversa.

## O que você pode criar

- **Mini apps web e ferramentas** — uma calculadora, um quiz, um formulário, uma pequena demonstração interativa.
- **Documentos** — textos estruturados que você pode refinar e exportar.
- **Visuais** — gráficos, diagramas e dashboards de dados simples.
- **Código** que você pode ler e executar.

## Por que é poderoso para quem não é desenvolvedor

Você pode construir algo *utilizável* — "me faça uma calculadora de gorjeta para um jantar em grupo", "um dashboard a partir deste CSV" — descrevendo-o e depois refinando-o de forma conversacional ("adicione um campo de taxa de serviço", "deixe os botões maiores"). É o exemplo mais claro de **construir com IA sem você mesmo escrever código**.

## Como trabalhar com artefatos

1. **Peça a coisa**, com especificidades (propósito, entradas, aparência).
2. **Itere em linguagem simples** — o Claude atualiza o mesmo artefato.
3. **Use-o** no painel; **exporte/compartilhe** onde houver suporte.

```mermaid
flowchart LR
  A[Describe what you want] --> B[Claude builds an artifact]
  B --> C[You try it in the side panel]
  C --> D[“Tweak this…”] --> B
```

## Dicas

- **Seja concreto** sobre entradas/saídas e público — assim como em um bom [prompt](/docs/prompting/basics).
- **Itere em pequenos passos.** Uma mudança de cada vez é mais fácil de acertar.
- **Verifique qualquer lógica/número** que um artefato calcule para usos importantes ([Alucinações](/docs/foundations/hallucinations)).
