Além de texto na tela, o Claude pode produzir **arquivos reais e baixáveis** — documentos do Word, apresentações do PowerPoint, planilhas do Excel, PDFs — formatados e prontos para usar. Ótimo quando você precisa de um entregável, não apenas de uma resposta.

## O que você pode pedir

- **Word (.docx)** — relatórios, cartas, propostas, documentos formatados.
- **PowerPoint (.pptx)** — apresentações de slides a partir de um esboço ou de anotações.
- **Excel (.xlsx)** — planilhas com dados, fórmulas, modelos simples.
- **PDF** — versões prontas para impressão dos itens acima.

## Como obter um bom arquivo

1. **Descreva o entregável**: tipo, público, extensão, seções, tom.
2. **Forneça o conteúdo ou a fonte** (anotações, dados, um esboço) — ou peça ao Claude para redigi-lo primeiro.
3. **Peça o arquivo** ("exporte isto como um .docx"), depois **itere** ("adicione uma capa", "transforme a tabela em um gráfico", "aperte o slide 3").

```text
Turn these notes into a 6-slide investor update deck (.pptx): traction, revenue,
roadmap, team, ask. Clean and minimal. Notes: {paste}
```

## Dicas

- **Dê estrutura.** Títulos/seções no seu pedido → saída mais limpa.
- **Forneça dados reais** para planilhas — e **verifique** quaisquer números calculados ([Alucinações](/docs/foundations/hallucinations)); não entregue valores que você não conferiu.
- **Refine antes de exportar** — é mais fácil corrigir no chat do que no arquivo final.
- Esses recursos são impulsionados por [Skills](/docs/claude-code/skills) por baixo dos panos — a mesma ideia que você mesmo pode criar.
