Um resumo em linguagem simples do que mudou recentemente em todo o Claude — para que você possa dar uma olhada em "o que há de novo e o que isso significa para mim" sem ler cada changelog. Mais recentes primeiro.

:::info Como esta página se mantém atualizada
As entradas são adicionadas pela comunidade e por um [observador automatizado](/docs/contribute/automation) que sinaliza mudanças upstream para que um humano as documente. Cada entrada aponta para sua fonte oficial. Notou algo faltando? [Adicione](/docs/contribute/contribute-in-10-minutes).
:::

## Como ler uma entrada

Cada atualização é etiquetada por **superfície** — 🗣️ Apps (chat/voz/mobile), 🛠️ Claude Code, 🔌 API, 🧠 Modelos — e responde **"o que isso significa para você."**

## Atualizações recentes

> _Entrada semente — substitua por atualizações reais e referenciadas conforme forem lançadas._

- 🧠 **Modelos** — Confira a tabela de [Modelos e Preços Atuais](/docs/whats-new/models-and-pricing) para a linha de modelos Claude e os IDs mais recentes. *O que isso significa:* comece novos projetos no padrão atual e procure os IDs exatos ali.
- 🛠️ **Claude Code** — Recursos são lançados com frequência (skills, hooks, MCP, agendamento). *O que isso significa:* se a data de `last verified` de uma página parecer antiga, confira o [changelog oficial](https://code.claude.com/docs/en/overview).
- 🔌 **API** — Capacidades como saída estruturada, extended thinking e prompt caching evoluem. *O que isso significa:* verifique novamente a [seção da API](/docs/api/first-call) antes de depender de parâmetros exatos.
