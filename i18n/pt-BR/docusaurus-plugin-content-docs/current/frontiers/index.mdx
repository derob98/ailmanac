# Fronteiras

A maior parte do AILmanac documenta como as coisas funcionam: recursos, configurações, preços, comandos. **Fronteiras** é diferente. É onde colocamos as **ferramentas de raciocínio** — as lentes que mudam como você toma decisões, não apenas quais botões aperta.

Dois tipos de conteúdo vivem aqui:

1. **Os modelos mentais do próprio AILmanac.** Enquadramentos originais que achamos genuinamente úteis e que não vimos empacotados em outro lugar. Somos transparentes: estas são *nossas* lentes opinativas, não fatos externos consolidados. Se uma delas finalmente fizer algo encaixar para você, esse é exatamente o objetivo.
2. **Ideias reais, mas subexploradas.** Conceitos que são verdadeiros e importantes, mas que a maioria dos guias ignora — geralmente porque não estão atrelados a um único recurso ou captura de tela.

Um bom modelo mental ganha seu valor de duas formas: ele sobrevive ao contato com o mundo real e se transfere. Os frameworks aqui deliberadamente **não** dependem de nenhuma versão de modelo ou preço específico. Essas coisas mudam toda semana (esse é literalmente o assunto da primeira página). As lentes foram feitas para durar mais do que elas.

## Como ler estas páginas

Trate cada página como uma ferramenta, não como uma doutrina. Pegue a parte que é útil, deixe o resto. Preferimos lhe dar um enquadramento que reorganize como você pensa do que dez dicas que você vai esquecer até a hora do almoço.

Se você identificar uma versão mais afiada de qualquer uma dessas — ou tiver uma lente própria que valha a pena compartilhar — é exatamente esse o tipo de coisa que queremos. Veja [Contribua em 10 Minutos](/docs/contribute/contribute-in-10-minutes).
