صندوق أدوات من أنماط كتابة المطالبات القابلة لإعادة الاستخدام. كل بطاقة: ما هي، ومتى تلجأ إليها، وقالب جاهز للنسخ واللصق. وهي قابلة للتركيب — اجمعها بحرية. جديد هنا؟ ابدأ بـ [أساسيات كتابة المطالبات](/docs/prompting/basics) أولاً.

## بدون أمثلة (اطلب فقط، بوضوح)
**متى:** المهام البسيطة والشائعة. **لماذا:** أقل جهد، وغالباً كافٍ.
```text
Summarize the text below in 5 bullets for a busy manager.
TEXT: """{paste}"""
```

## أمثلة قليلة (اعرض أمثلة)
**متى:** عندما تحتاج إلى صيغة أو أسلوب محدد. **لماذا:** الأمثلة تعلّم أسرع من الأوصاف. انظر [الأمثلة القليلة بشكل صحيح](/docs/prompting/few-shot).
```text
Classify the sentiment as positive/negative/neutral.
Review: "Loved it, shipped fast" → positive
Review: "Broke on day one" → negative
Review: "{new review}" →
```

## سلسلة التفكير (فكّر خطوة بخطوة)
**متى:** الرياضيات والمنطق والاستدلال متعدد الخطوات. **لماذا:** الاستدلال قبل الإجابة يحسّن الدقة.
```text
Work through this step by step, then give the final answer on its own line.
PROBLEM: {problem}
```

## التفكيك (قسّمها)
**متى:** المهام الكبيرة والغامضة. **لماذا:** المهام الفرعية الأصغر أسهل وأكثر موثوقية.
```text
First, list the sub-tasks needed to {goal}. Then do them one at a time,
showing each result before moving on.
```

## مطالبة الدور (اسند شخصية)
**متى:** عندما تريد منظوراً أو معياراً محدداً. **لماذا:** تهيّئ السلوك المناسب.
```text
You are a meticulous senior copy editor. Edit the text for clarity and concision
without changing meaning. Show a bullet list of the changes you made.
```

## قيد صيغة المخرجات
**متى:** عندما تتغذى النتيجة على أداة أخرى، أو عندما تريد الاتساق. **لماذا:** الأشكال القابلة للتوقع قابلة للاستخدام.
```text
Reply with ONLY valid JSON matching: {"title": string, "tags": string[]}.
No prose, no markdown fences.
```

## الاتساق الذاتي / النقد الذاتي
**متى:** عندما تهم الصحة. **لماذا:** المرور الثاني يلتقط أخطاء المرور الأول.
```text
Answer the question. Then, in a section called "Check", critique your own answer
for errors or missing cases and revise if needed.
```

## المطالبة الفوقية (دع الذكاء الاصطناعي يحسّن مطالبتك)
**متى:** عندما يكون أداء مطالبتك ضعيفاً. **لماذا:** النموذج بارع في رصد ما ينقصه.
```text
Here's my prompt: "{prompt}". Ask me the 3 questions that would most improve it,
then rewrite it incorporating sensible defaults.
```

## التحسين التكراري
**متى:** دائماً تقريباً. **لماذا:** المخرَج الأول مسودة، وليس الوجهة النهائية.
```text
{after a result} Shorter and more concrete. Keep the second paragraph. Add one
example. Drop the marketing tone.
```

:::tip اجمعها معاً
المطالبة الحقيقية القوية غالباً ما تكون: **دور** + **مثال** + **قيد على المخرجات** + إذن بقول **"لا أعرف."** اجمع البطاقات طبقات.
:::
