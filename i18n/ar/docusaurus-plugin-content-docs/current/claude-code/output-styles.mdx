يمكن أن تتغير أنماط المخرَجات المتاحة وطريقة ضبطها — تأكد من ذلك في الوثائق الرسمية.
</VerifyNote>

تغيّر **أنماط المخرَجات (output styles)** *كيف* يتواصل Claude Code — نبرته، وإسهابه، وما إذا كان يشرح استدلاله — دون تغيير *ما* يستطيع فعله. اعتبرها إعدادًا مسبقًا للشخصية/التنسيق للردود.

## لماذا تستخدمها

- **طابِق سير عملك.** مقتضب وموجّه للفعل عندما تتحرك بسرعة؛ وشارح عندما تتعلم.
- **وضع التعليم.** تضيف بعض الأنماط تعليقًا تعليميًا — رائع وأنت تتعرف على قاعدة شيفرة أو تقنية.
- **الاتساق.** اضبط نمطًا مفضّلًا فتُقرأ كل جلسة بالطريقة التي تحبها.

## الأنماط الشائعة

تتراوح الأنماط المدمجة النمطية من نمط افتراضي موجز إلى أوضاع أكثر **شرحًا** أو **توجّهًا نحو التعلّم** تمر عبر "اللماذا". اضبط النمط النشط عبر الأمر/الإعداد ذي الصلة (راجع [الوثائق الرسمية](https://docs.anthropic.com/en/docs/claude-code/output-styles) للقائمة الحالية والآلية المحددة).

## أنماط المخرَجات مقابل CLAUDE.md مقابل أنماط الرد

- **نمط المخرَجات** = إعداد مسبق للعرض لردود Claude Code (النبرة/الإسهاب).
- **[CLAUDE.md](/docs/claude-code/claude-md)** = حقائق وقواعد *المشروع* الدائمة (ماذا تفعل، الأعراف، الحواجز الواقية).
- **[التعليمات المخصصة / أنماط الرد](/docs/claude-app/custom-instructions)** = النظير في تطبيق Claude.ai الاستهلاكي.

:::tip استخدم الرافعة الصحيحة
تريد من Claude أن "يشغّل الاختبارات دائمًا قبل الانتهاء"؟ هذا **حاجز واقٍ في CLAUDE.md**، لا نمط. تريد منه أن "يشرح كل خطوة وهو يمضي"؟ هذا **نمط مخرَجات**. أبقِ قواعد السلوك في CLAUDE.md والعرض في الأنماط.
:::
