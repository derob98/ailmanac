بناء تجارب شبيهة بالدردشة على واجهة برمجة التطبيقات يتلخص في حقيقتين عمليتين: **بثّ** الردود حتى يرى المستخدمون المخرجات فورًا، و**إدارة السجل بنفسك** لأن واجهة برمجة التطبيقات عديمة الحالة. أتقِن كليهما وستشعر تجربة الدردشة لديك بأنها سريعة وتتذكر كل شيء.

## البث

من دون البث، ينتظر المستخدم الرد بأكمله. مع البث، تصل الرموز فور توليدها — وهو ما يحسّن السرعة المدرَكة بشكل كبير.

<Callout type="tip" items={["استخدم مساعد البث في حزمة التطوير (SDK) بدلًا من تحليل الأحداث الخام يدويًا — فهو يدير دورة حياة الأحداث نيابةً عنك."]} />

<Tabs groupId="lang">
<TabItem value="python" label="Python">

```python
with client.messages.stream(
    model="claude-sonnet-5", max_tokens=1024,
    messages=[{"role": "user", "content": "Explain RAG in two sentences."}],
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)
```

</TabItem>
<TabItem value="ts" label="TypeScript">

```ts
const stream = client.messages.stream({
  model: "claude-sonnet-5", max_tokens: 1024,
  messages: [{ role: "user", content: "Explain RAG in two sentences." }],
});
for await (const event of stream) {
  if (event.type === "content_block_delta") process.stdout.write(event.delta.text ?? "");
}
```

</TabItem>

## متعدد الأدوار: أنت من يحمل السجل

ليس لدى واجهة برمجة التطبيقات **أي ذاكرة** بين الاستدعاءات ([لماذا](/docs/foundations/tokens-and-context)). لمتابعة محادثة، أرسِل **التبادل السابق بأكمله** في كل مرة.

<Steps items={[{title: "ابدأ المحادثة", body: "أرسِل أول رسالة من المستخدم ضمن قائمة الرسائل."}, {title: "التقِط الرد", body: "اقرأ نص رد المساعد من النتيجة."}, {title: "أضِف كلا الدورين", body: "أضِف رد المساعد ورسالة المستخدم التالية إلى قائمة الرسائل نفسها."}, {title: "أعِد إرسال القائمة الكاملة", body: "أرسِل قائمة الرسائل الكاملة مرة أخرى — هكذا 'يتذكر' Claude التبادل."}]} />

```python
messages = [{"role": "user", "content": "Hi, I'm planning a trip."}]
# ... get assistant reply, then append both turns:
messages.append({"role": "assistant", "content": assistant_text})
```

## المحادثات الطويلة تملأ النافذة

كلما نما السجل، استهلك [نافذة السياق](/docs/foundations/tokens-and-context) وارتفعت التكلفة. إليك استراتيجيات للسيطرة عليه:

- **التلخيص/الضغط** للأدوار الأقدم في موجز قصير تحمله معك.
- **التشذيب** للأدوار السابقة غير ذات الصلة.
- اقترِنها مع **[التخزين المؤقت للموجّهات](/docs/api/prompt-caching)** لتجنّب الدفع مجددًا مقابل بادئة ثابتة.
