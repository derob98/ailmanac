تتيح الواجهة البرمجية لبرنامجك *أنت* أن يتحدّث إلى Claude. ثلاث خطوات: احصل على مفتاح، وثبّت حزمة SDK، وأرسل رسالة.

## 1. احصل على مفتاح API

أنشئ واحدًا في [وحدة تحكّم Anthropic](https://console.anthropic.com). ثم اضبطه كمتغيّر بيئة كي لا يظهر أبدًا في شيفرتك:

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

:::warning لا تُودِع مفتاحك في نظام التحكّم بالإصدارات أبدًا
احتفظ بالمفاتيح في متغيّرات البيئة أو في مدير أسرار — لا في نظام التحكّم بالشيفرة المصدرية إطلاقًا. راجع [الأمان](/docs/security).
:::

## 3. نفّذ الاستدعاء

كل طلب هو قائمة من `messages`. ويردّ النموذج بـ `content`.

<Tabs groupId="lang">
<TabItem value="python" label="Python">

```python
import anthropic

client = anthropic.Anthropic()  # reads ANTHROPIC_API_KEY from the environment

message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "In one sentence, what is an API?"}
    ],
)

print(message.content[0].text)
```

</TabItem>
<TabItem value="ts" label="TypeScript">

```ts
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic(); // reads ANTHROPIC_API_KEY from the environment

const message = await client.messages.create({
  model: "claude-sonnet-4-6",
  max_tokens: 1024,
  messages: [
    { role: "user", content: "In one sentence, what is an API?" },
  ],
});

console.log(message.content[0].text);
```

</TabItem>
<TabItem value="curl" label="cURL">

```bash
curl https://api.anthropic.com/v1/messages \
  --header "x-api-key: $ANTHROPIC_API_KEY" \
  --header "anthropic-version: 2023-06-01" \
  --header "content-type: application/json" \
  --data '{
    "model": "claude-sonnet-4-6",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "In one sentence, what is an API?"}
    ]
  }'
```

</TabItem>
</Tabs>

## ماذا حدث للتو

- **`model`** — أيّ نموذج Claude تستخدم. لا تُرمّزه يدويًا بشكل أعمى؛ راجع [اختيار نموذج](/docs/api/choosing-a-model).
- **`max_tokens`** — حدّ أقصى لطول الردّ (بـ[الرموز](/docs/foundations)). إنه *لا* يحدّد نافذة السياق.
- **`messages`** — المحادثة حتى اللحظة. الواجهة البرمجية **عديمة الحالة**: لمتابعة محادثة، أعِد إرسال السجلّ الكامل في كل مرة.
- **`system`** (اختياري) — تعليمة على المستوى الأعلى تحدّد دور Claude في الاستدعاء.
