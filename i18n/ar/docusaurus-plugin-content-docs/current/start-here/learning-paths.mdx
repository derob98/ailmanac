اختر المسار الذي يشبهك أكثر. كل مسار هو طريق مرتّب — اتبعه بالترتيب، وتخطَّ ما تعرفه بالفعل. الأوقات تقريبية.

## 🌱 الوافد الفضولي · ~45 دقيقة

تريد استخدام الذكاء الاصطناعي جيدًا دون أن تصبح مبرمجًا.

1. [أول 5 دقائق لك](/docs/start-here/your-first-5-minutes) <LevelBadge level="beginner" />
2. [أي إصدار من Claude ينبغي أن أستخدم؟](/docs/start-here/which-claude) <LevelBadge level="beginner" />
3. [أساسيات كتابة الموجِّهات](/docs/prompting/basics) <LevelBadge level="beginner" />
4. [ما هو نموذج اللغة الكبير (LLM)؟](/docs/foundations) <LevelBadge level="beginner" />
5. [الإنتاجية اليومية](/docs/playbooks/productivity) <LevelBadge level="beginner" />

## 💼 العامل المعرفي · ~ساعتان

تريد للذكاء الاصطناعي أن يجعل عملك الحقيقي أسرع — الكتابة، والبحث، والتحليل.

1. مسار الوافد الفضولي (أعلاه)، ثم…
2. [مكتبة أنماط الموجِّهات](/docs/prompting) <LevelBadge level="all" />
3. [الهلوسات وكيفية الحد منها](/docs/foundations) <LevelBadge level="intermediate" />
4. دليل العمل الخاص بك: [الكتابة](/docs/playbooks)، أو [البحث](/docs/playbooks)، أو [تحليل البيانات](/docs/playbooks)
5. [المشاريع: مساحات عمل دائمة](/docs/claude-app) و[قوالب الموجِّهات القابلة لإعادة الاستخدام](/docs/templates)

## 🛠️ الباني / المطوّر · ~نصف يوم

تكتب الشيفرة وتريد تخصيص Claude Code والبناء على واجهة API.

1. [ما هو Claude Code](/docs/claude-code/what-is-claude-code) <LevelBadge level="beginner" />
2. [CLAUDE.md وملفات الذاكرة](/docs/claude-code) <LevelBadge level="beginner" /> + [وضع التخطيط (Plan Mode)](/docs/claude-code)
3. [الأذونات](/docs/claude-code) و[settings.json](/docs/claude-code) <LevelBadge level="intermediate" />
4. [أول استدعاء لك لواجهة API](/docs/api/first-call) ← [استخدام الأدوات (Tool Use)](/docs/api) ← [بناء الوكلاء](/docs/api) <LevelBadge level="advanced" />
5. [شرح تطبيقي: خصّص Claude Code لمستودع حقيقي](/docs/walkthroughs) <LevelBadge level="intermediate" />
6. [الأمان والاستخدام المسؤول](/docs/security) <LevelBadge level="advanced" />

## 👥 قائد الفريق · ~3 ساعات

أنت تطرح الذكاء الاصطناعي على فريق وتهتم بالاتساق والأمان والتكلفة.

1. أساسيات مسار الباني (1–3 أعلاه)
2. [الإضافات والأسواق (Plugins & Marketplaces)](/docs/claude-code) — أطلق مجموعة أدوات مشتركة <LevelBadge level="advanced" />
3. [مقايضات التكلفة والكمون (Latency)](/docs/foundations) و[اختيار نموذج](/docs/api/choosing-a-model) <LevelBadge level="intermediate" />
4. [تأمين الوكلاء](/docs/security) و[تحصين عمليات التشغيل المستقلة](/docs/security) <LevelBadge level="advanced" />
5. [الاستخدام المسؤول والتحقق](/docs/security) — ضع أعراف الفريق <LevelBadge level="all" />
