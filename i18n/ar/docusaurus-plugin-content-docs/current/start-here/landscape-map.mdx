لا يوجد AILmanac في فراغ، ولن نتظاهر بأننا نستبدل كل شيء. إليك خريطة صادقة لأفضل المصادر — ومتى تستخدمها **بدلًا منا**.

## مصدر الحقيقة

- **[وثائق Anthropic](https://docs.anthropic.com)** — المرجع الموثوق لـ Claude وواجهة API والنماذج والميزات. **عندما يختلف AILmanac مع الوثائق، فالوثائق هي الفيصل.** استخدمها للتفاصيل الدقيقة والحالية؛ واستخدمنا للإرشاد ذي الرأي الواضح، المتدرج، القائم على التعلّم بالممارسة.

## مصادر Anthropic الرسمية

- **[مكتبة الموجِّهات (Prompt Library)](https://docs.anthropic.com/en/resources/prompt-library/library)** — موجِّهات مثال جاهزة.
- **[كتاب وصفات Anthropic (Cookbook)](https://github.com/anthropics/anthropic-cookbook)** — وصفات API قابلة للتشغيل (استخدام الأدوات، وRAG، والرؤية).
- **[وثائق Claude Code](https://code.claude.com/docs/en/overview)** — مرجع CLI/IDE.
- **[أكاديمية Anthropic / Learn](https://www.anthropic.com/learn)** — دورات إرشادية مجانية.
- **[بروتوكول سياق النموذج (Model Context Protocol)](https://modelcontextprotocol.io)** — معيار MCP المفتوح + دليل الخوادم.

## المجتمع والعام

- **[awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code)** — أوامر وخطّافات (hooks) وسير عمل منتقاة.
- **[دليل هندسة الموجِّهات (Prompt Engineering Guide)](https://www.promptingguide.ai)** — مرجع عميق ومحايد تجاه المزوّدين لكتابة الموجِّهات.
- **[Learn Prompting](https://learnprompting.org)** — دورة منظمة وودودة للمبتدئين لكتابة الموجِّهات.

## متى تستخدم ماذا

| تريد… | اذهب إلى |
|---|---|
| تفاصيل API/ميزة دقيقة وحالية | الوثائق الرسمية |
| موجِّهات مثال للنسخ واللصق | مكتبة الموجِّهات / [قوالبنا](/docs/templates/prompts) |
| شيفرة API قابلة للتشغيل | كتاب الوصفات (Cookbook) |
| مسار ذو رأي واضح ومصنَّف بالمستوى | **AILmanac** |
| مرجع نظري عميق عن كتابة الموجِّهات | promptingguide.ai |
| دورة إرشادية | أكاديمية Anthropic |

## أين يقع AILmanac

نحن **الدليل الميداني ذو الرأي الواضح**: الطريقة الموصى بها أولًا، مصنَّفة بالمستوى، غنية بالقوالب، صادقة بشأن الحداثة، وتمتد عبر Claude **و**مفاهيم الذكاء الاصطناعي العامة. اعتبرنا الخريطة التي تشير إلى المصدر العميق المناسب في الوقت المناسب — وتعلّم الأساسيات حتى تصبح كلها منطقية.
