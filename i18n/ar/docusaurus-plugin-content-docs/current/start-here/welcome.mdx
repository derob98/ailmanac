**AILmanac** هو تقويم مرجعي مجاني بناه المجتمع **لتحقيق أقصى استفادة من Claude — ومن أي ذكاء اصطناعي**. التقويم المرجعي هو مرجع تعود إليه مرارًا: شامل، ومكتوب بلغة بسيطة، ومُحدَّث باستمرار. هذا هو الهدف هنا.

## لمن هذا الموقع

للجميع، عن قصد — ونخبرك بمستوى كل صفحة حتى لا تشعر أبدًا بالتيه أو بأننا نتحدث إليك بتعالٍ:

| المستوى | هذا أنت إذا... |
| --- | --- |
| <LevelBadge level="beginner" /> | بالكاد استخدمت الذكاء الاصطناعي وتريد فوزًا أوّلًا مضمونًا. |
| <LevelBadge level="intermediate" /> | تستخدم Claude بالفعل وتريد نتائج أفضل بشكل ملحوظ. |
| <LevelBadge level="advanced" /> | تبني باستخدام Claude Code وواجهة API والوكلاء وMCP. |

## ما الذي يجعل هذا مختلفًا

هناك بالفعل مصادر ممتازة (انظر [خريطة المشهد العام](/docs/start-here/landscape-map)). إن AILmanac **ليس** مجرد قائمة روابط و**ليس** نسخة من الوثائق الرسمية. إنه:

- **ذو رأي واضح** — نخبرك *بالطريقة الموصى بها الواحدة*، ثم البدائل.
- **مُصنَّف بالمستوى** — كل صفحة تحمل شارة لتقرأ ما يناسبك.
- **غني بالقوالب** — [نماذج بداية جاهزة للنسخ واللصق](/docs/templates) يمكنك استخدامها في 30 ثانية.
- **صادق بشأن الحداثة** — الحقائق التي تتغير (أسماء النماذج، الأسعار، الميزات) تحمل تاريخ "آخر تحقق" ورابطًا إلى المصدر.

## ما الذي ليس عليه هذا الموقع

:::warning الوثائق الرسمية هي مصدر الحقيقة
عندما يختلف AILmanac مع [وثائق Anthropic الرسمية](https://platform.claude.com/docs)، فإن **الوثائق الرسمية هي الفيصل**. تتغير أدوات الذكاء الاصطناعي بسرعة؛ نعمل جاهدين لمواكبة المستجدات، لكن تحقق دائمًا من أي أمر بالغ الأهمية مقابل المصدر الذي نربط إليه.
:::

كذلك لا نخزّن بياناتك، ولا يمكننا رؤية حسابك، ولن نتخذ القرارات نيابةً عنك في الأعمال عالية المخاطر — انظر [الاستخدام المسؤول](/docs/security).
