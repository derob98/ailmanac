## 1. Ottieni una chiave API

Creane una nella [Anthropic Console](https://console.anthropic.com). Poi impostala come variabile d'ambiente così che non finisca mai nel tuo codice:

<PromptCard title="Esporta la tua chiave API">{`export ANTHROPIC_API_KEY="sk-ant-..."`}</PromptCard>

<Callout type="warning" items={["Non committare mai la tua chiave. Conserva le chiavi in variabili d'ambiente o in un secrets manager, mai nel controllo di versione. Vedi la pagina sulla Sicurezza (/docs/security)."]} />

## 2. Installa l'SDK

Scegli il tuo linguaggio. L'SDK legge la chiave automaticamente dalla variabile d'ambiente che hai appena impostato.

<Tabs groupId="lang">
<TabItem value="python" label="Python">

<PromptCard title="Installazione (Python)">{`pip install anthropic`}</PromptCard>

</TabItem>
<TabItem value="ts" label="TypeScript">

<PromptCard title="Installazione (TypeScript)">{`npm install @anthropic-ai/sdk`}</PromptCard>

</TabItem>
<TabItem value="curl" label="cURL">

Nulla da installare — ti serve solo `curl`.

</TabItem>
</Tabs>

## 3. Esegui la chiamata

Ogni richiesta è una lista di `messages`. Il modello risponde con `content`. Esegui lo snippet per il tuo linguaggio e vedrai Claude rispondere alla domanda.

<Tabs groupId="lang">
<TabItem value="python" label="Python">

```python
import anthropic

client = anthropic.Anthropic()  # reads ANTHROPIC_API_KEY from the environment

message = client.messages.create(
    model="claude-sonnet-5",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "In one sentence, what is an API?"}
    ],
)

print(message.content[0].text)
```

</TabItem>
<TabItem value="ts" label="TypeScript">

```ts
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic(); // reads ANTHROPIC_API_KEY from the environment

const message = await client.messages.create({
  model: "claude-sonnet-5",
  max_tokens: 1024,
  messages: [
    { role: "user", content: "In one sentence, what is an API?" },
  ],
});

console.log(message.content[0].text);
```

</TabItem>
<TabItem value="curl" label="cURL">

```bash
curl https://api.anthropic.com/v1/messages \
  --header "x-api-key: $ANTHROPIC_API_KEY" \
  --header "anthropic-version: 2023-06-01" \
  --header "content-type: application/json" \
  --data '{
    "model": "claude-sonnet-5",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "In one sentence, what is an API?"}
    ]
  }'
```

</TabItem>
</Tabs>

## Cosa è appena successo

Hai inviato quattro informazioni. Ecco cosa controlla ciascuna:

- **`model`** — quale Claude usare. Non scriverlo a mano alla cieca; vedi [Scegliere un modello](/docs/api/choosing-a-model).
- **`max_tokens`** — un limite sulla lunghezza della risposta (in [token](/docs/foundations)). *Non* imposta la finestra di contesto.
- **`messages`** — la conversazione finora. L'API è **stateless**: per continuare una chat, devi rinviare l'intera cronologia ogni volta.
- **`system`** (opzionale) — un'istruzione di alto livello che definisce il ruolo di Claude per la chiamata.
