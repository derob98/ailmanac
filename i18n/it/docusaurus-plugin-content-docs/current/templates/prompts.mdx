Prompt pronti all'uso per le attività comuni. Usano i [tag XML](/docs/prompting/xml-tags) per una struttura pulita e incorporano le abitudini illustrate in [Le basi del prompting](/docs/prompting/basics). Compila le `{parentesi graffe}`.

## Riassumere (ancorato)

```text
<instructions>
Summarize the document for {audience} in {N} bullet points. Use ONLY the document;
if something isn't there, say "not stated." Then list any action items.
</instructions>
<document>
{paste}
</document>
```
*Atteso:* punti elenco concisi + elementi d'azione. *Fallimento comune:* inventa fatti → la clausola "ONLY the document" lo previene.

## Estrarre dati strutturati

```text
Extract the following from the text as JSON: {field1, field2, field3}.
Return ONLY valid JSON, no prose. If a field is missing, use null.
<text>{paste}</text>
```
*Abbinalo all'*[output strutturato](/docs/api/structured-output) per maggiore affidabilità.

## Riscrivere / modificare

```text
Rewrite the text to be {clearer/shorter/more formal} without changing meaning.
Then list the changes you made as bullets.
<text>{paste}</text>
```

## Revisione del codice

```text
Review this diff for correctness bugs, missing edge cases, and security issues.
Report ONLY high-confidence findings as a checklist with file:line. Diff:
<diff>{paste}</diff>
```

## Assistente alle decisioni

```text
I'm deciding between {A} and {B} for {situation}. Build a trade-off table across
the criteria that matter, then recommend one and name the biggest risk.
```

## Meta-prompt (migliorare un prompt)

```text
Here's my prompt: "{prompt}". Ask me the 3 questions that would most improve it,
then rewrite it with sensible defaults.
```

## Wrapper con ruolo + guardrail

```text
You are {role}. {task}.
Constraints: {format, length, tone}. If you're unsure or lack info, say so rather
than guessing.
```

Salva quelli che usi come [comandi slash](/docs/templates/slash-commands) o come un [Progetto](/docs/claude-app/projects) così non dovrai riscriverli mai più.
:::
