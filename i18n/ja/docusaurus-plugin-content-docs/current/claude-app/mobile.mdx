Claudeはデスクトップのタブだけのものではありません。**iOSおよびAndroidアプリ**がClaudeをあなたのポケットに入れてくれます。アイデアの記録、ちょっとした質問、外出先での[音声](/docs/claude-app/voice-mode)に最適です。

## モバイルが得意なこと

- **移動中の記録。** エレベーターでの思いつき、ホワイトボードの写真、サッと「これを要約して」 — そうでなければ失われてしまう入力に、モバイルは真価を発揮します。
- **音声ファースト。** スマホでは話すほうが入力より速いことが多いです。[音声モード](/docs/claude-app/voice-mode)を参照してください。
- **写真＆スクリーンショット。** メニュー、フォーム、エラーメッセージ、看板を撮影して質問できます（マルチモーダル入力）。
- **継続性。** 会話や[プロジェクト](/docs/claude-app/projects)が同期されるので、モバイルで始めてデスクトップで仕上げられます。

## デスクトップのほうが優れていること

- 長文の**ライティングと編集**。
- **大きなドキュメント**や多数のファイルを扱う作業。
- **[Artifacts](/docs/claude-app/artifacts)**や、画面の広さが役立つあらゆるもの。
- **開発者**の作業（[Claude Code](/docs/claude-code/what-is-claude-code)はデスクトップ／CLIです）。

## 便利なモバイルワークフロー

**音声で記録 → 画面で推敲。** 歩きながら[音声](/docs/claude-app/voice-mode)で散らかったアイデアを吐き出し、後でClaudeに*「あの文字起こしをきれいなアウトラインにして」*と頼みます。スマホが生の素材を集め、あなたはどこででも磨きをかけます。

:::tip 共有シートを使う
モバイルでは、他のアプリからテキスト／リンク／画像を直接Claudeに共有できることがよくあります。コピー＆ペーストせずに「これを要約して」と頼む素早い方法です。
:::
