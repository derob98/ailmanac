## 目的別のティア

| ティア | おおよその対象 |
|---|---|
| **Free** | Claudeを試す、軽い個人利用 |
| **Pro** | 日常的な個人利用 — より多くの利用量、より多くの機能 |
| **Max** | Proの上限に達するヘビーな個人ユーザー |
| **Team** | 小規模チーム — 共有ワークスペースと管理機能 |
| **Enterprise** | 管理、セキュリティ、スケール、コントロールを必要とする組織 |

（正確な名称や構成は変わることがあります。価格ページで確認してください。）

## 「制限」とはどのように感じるか

コンシューマープランは**一定期間内にどれだけ使えるか**に上限を設けており（時間とともにリセットされるメッセージ数／利用量）、上位ティアほどその上限が引き上げられます。上限に達すると、通常はリセットされるまで待つか、アップグレードするよう促されます。作業内容は失われず、一時停止するだけです。

重いモデル（そして長い会話）ほど、利用枠を速く消費します。上限に達しがちな場合の手早い対処法は、新しいトピックには**新しいチャットを始める**こと、そしてプロンプトを焦点の絞られたものに保つことです（[コンテキスト管理](/docs/foundations/tokens-and-context)と同じ考え方）。

## 選び方

1. **Freeから始める。** 実際に何に使うのかを把握しましょう。
2. 制限が実務を妨げるようになったら**Proにアップグレード**します。
3. パワーユーザーでまだ上限に達するなら**Maxへ**。
4. 同僚が共有アクセス、管理、コンプライアンスを必要とするなら**Team／Enterprise**へ。

## アプリ課金とAPI課金 — 別物です

- **アプリプラン** ＝ 利用上限付きの定額サブスクリプション（このページ）。
- **API課金** ＝ 構築したものに対して[トークン](/docs/api/tokens-and-pricing)単位で支払う。別のモデル、別のページです。
