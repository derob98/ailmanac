画面上のテキストにとどまらず、Claudeは**実際にダウンロードできるファイル**、つまりWordドキュメント、PowerPointのスライド、Excelのスプレッドシート、PDFを、書式付きですぐ使える状態で作成できます。単なる回答ではなく成果物が必要なときに最適です。

## 依頼できるもの

- **Word（.docx）** — レポート、手紙、提案書、書式付きドキュメント。
- **PowerPoint（.pptx）** — アウトラインやメモからのスライドデッキ。
- **Excel（.xlsx）** — データ、数式、シンプルなモデルを含むスプレッドシート。
- **PDF** — 上記の印刷可能なバージョン。

## 良いファイルを得る方法

1. **成果物を説明する** — 種類、対象読者、長さ、セクション、トーン。
2. **コンテンツやソースを提供する**（メモ、データ、アウトライン） — もしくはClaudeにまず下書きを依頼します。
3. **ファイルを依頼する**（「これを.docxとしてエクスポートして」）、そして**改善する**（「タイトルページを追加して」「表をチャートにして」「3枚目のスライドを引き締めて」）。

```text
Turn these notes into a 6-slide investor update deck (.pptx): traction, revenue,
roadmap, team, ask. Clean and minimal. Notes: {paste}
```

## ヒント

- **構造を与える。** 依頼に見出しやセクションを入れる → よりきれいな出力になります。
- スプレッドシートには**実データを提供**し、計算された数値はすべて**検証**してください（[ハルシネーション](/docs/foundations/hallucinations)）。確認していない数字をそのまま出さないようにしましょう。
- **エクスポート前に推敲する** — 最終ファイルよりチャット内で直すほうが簡単です。
- これらの機能は裏側で[Skills](/docs/claude-code/skills)によって動いています。あなた自身が作れるのと同じ仕組みです。
