最も速い入力があなたの声であることもあります。音声モードを使うと、Claudeに**話しかけて**、その返答を聞いたり読んだりできます。アイデアの記録、声に出して考えること、Claudeのハンズフリー利用に最適です。

## 何に向いているか

- **移動中のアイデアの記録** — 散らかった思いつきを語り、Claudeにそれを構造化してもらいます。
- **ハンズフリーの場面** — 歩行中、通勤中、料理中。
- **練習やブレインストーミング** — 声に出すと、やり取りが自然に感じられます。
- **アクセシビリティ** — 多くの人にとって、話すほうが入力より簡単なことがあります（[アクセシビリティ](/docs/playbooks/accessibility)を参照）。

## 始め方

1. Claudeアプリを開きます（通常はモバイルアプリ。提供状況は異なります。上の注記を参照してください）。
2. チャット画面から音声会話を開始します。
3. 自然に話し、終わったら一拍おきます。Claudeが応答し、後で見返したりコピーしたりできるよう、**文字起こし**がチャットに保存されます。
4. 通常は**プッシュ・トゥ・トーク**と**ハンズフリー**のスタイルを切り替えられ、**多言語対応**です。あなたが考える言語で話してください。

## 良い音声結果を得るためのヒント

- **依頼を冒頭に置く。** 「結婚式のスピーチのアウトラインを手伝って — 3分、温かくユーモラスに」のほうが、だらだら話すよりうまくいきます。
- **欲しい形式を言う。** 「箇条書きで5つ挙げて」 — そうしないと、話し言葉の返答はとりとめなくなりがちです。
- **下書きには音声、推敲には画面を使う。** 音声で記録し、同じチャット内でテキストを整えましょう。

:::tip 音声で入力、テキストで出力
強力なワークフロー: 音声で頭の中を吐き出し、それから*「あれをきれいなメールにして」*と入力します。文字起こしがあなたの生の素材になります。
:::
