# フロンティア

AILmanacの大部分は、機能・設定・価格・コマンドといった「仕組み」を記録しています。**フロンティア**はそれとは違います。ここに置くのは**思考の道具** — どのボタンを押すかではなく、どう意思決定するかを変えるレンズです。

ここには2種類のコンテンツがあります。

1. **AILmanac独自のメンタルモデル。** 私たちが本当に役立つと感じ、他では体系化されているのを見たことのないオリジナルの捉え方です。これらが確立された外部の事実ではなく、*私たち*の主観的なレンズであることは正直にお伝えします。そのひとつであなたの中で何かがようやく腑に落ちたなら、それこそが狙いです。
2. **実在するのに十分活用されていないアイデア。** 真実かつ重要なのに、ほとんどのガイドが見落としている概念です。たいていは、特定の機能やスクリーンショットに結びついていないからです。

優れたメンタルモデルは2つの形で役立ちます。現実との接触に耐え、そして転用が利く、という点です。ここで紹介するフレームワークは、特定のモデルのバージョンや価格について語るものでは**ありません**。そうしたものは毎週変わります(それこそが最初のページの主題そのものです)。レンズはそれらより長く生き残ることを意図しています。

## これらの読み方

各ページは教義ではなく道具として扱ってください。役立つ部分を取り、残りは置いていって構いません。昼食までに忘れてしまう10個のヒントよりも、あなたの考え方を組み替えるひとつの捉え方を渡したいと考えています。

これらのいずれかについてより鋭いバージョンを見つけたら — あるいは共有する価値のある独自のレンズをお持ちなら — それこそまさに私たちが求めているものです。[10分で貢献する](/docs/contribute/contribute-in-10-minutes)をご覧ください。
