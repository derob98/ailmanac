創業者や小規模チームにとって、AIはレバレッジです。ジュニアのマーケター、アナリスト、アシスタントの仕事を一度にこなします。うまく指示し、重要なことについては判断を握り続けてください。

## レバレッジの高い用途

- **ポジショニングとメッセージング** — 価値提案を固め、そこからコピーを導き出す。
- **マーケティングコンテンツ** — ランディングページ、メール、広告、ソーシャル、FAQ([ライティング](/docs/playbooks/writing)を参照)。
- **顧客コミュニケーション** — 返信の起草、クレーム対応、お知らせの作成。
- **リサーチ** — 市場/競合のスキャン(検証を!)、顧客インタビューの質問セット。
- **計画** — 目標を、優先順位付けされ時間枠の区切られた計画に変える。
- **ドキュメントと業務** — SOP、求人投稿、会議の要約。

## ポジショニングから始める

どんなコピーよりも先に、一言の表現を正しく決めましょう:

```text
Write 3 versions of a value proposition using: for {audience} who {need},
{product} is a {category} that {key benefit}, unlike {alternative}.
Then critique each in one line.
```

これが鋭くなれば、その下流のすべて(広告、メール、ランディングページ)が楽になります。

## ビジネス利用における注意点

:::warning 証拠をでっち上げず、主張に注意する
- 推薦の声、指標、ケーススタディ、受賞歴を**決してでっち上げない** — `[insert real number]` のプレースホルダーを使い、実際のデータで埋めてください。
- **規制対象または裏付けのない主張に注意する**(健康、金融、「No.1」、保証)。
- 行動する前に市場の事実と数値を**検証する**([ハルシネーション](/docs/foundations/hallucinations))。
- **データを守る** — 顧客の個人情報や機密の数値を、検証していないツールに貼り付けないでください([プライバシー](/docs/foundations/privacy))。
:::
