AIの最も意義深い用途の1つが、最も注目されていません。それは**支援ツール**としての用途です。Claudeは情報を*あなた*に合わせて適応できます — あなたのペース、あなたの形式、あなたの言語に。これは本当に状況を平等にしうるものです。

## 役立つ方法

- **ディスレクシア / 読書の負荷** — 「これを短く簡単な文に書き換えて」「要点を要約して」「これを平易な言葉で説明して」。
- **ロービジョン** — [音声モード](/docs/claude-app/voice-mode)と組み合わせて、聞いて口述する。スクリーンリーダーがうまく扱える、きれいで構造化されたテキストを求めてください。
- **神経多様性 / 集中(ADHDなど)** — 「これをチェックリストに分解して」「次の一手は何?」、文章の壁を明確な計画に変える、行き詰まっているメールを起草する。
- **語学学習者** — 翻訳、簡略化、慣用句の説明、会話の練習、または同じ考えをさまざまな難易度で得る。
- **書くことへの不安** — 反応するための判断のない初稿を得て、始めるハードルを下げる。

## 試すプロンプト

```text
Rewrite this so it's easy to read: short sentences, simple words, one idea per
line. Keep all the meaning. Text: {paste}
```
```text
Turn this into a step-by-step checklist with the single most important step
marked. {paste}
```
```text
Explain this in {your language}, then give me 3 phrases to practice.
```

## 心に留めておくこと

- **これは補助であって、臨床医ではありません。** 健康/医療/法律上の決定については専門家に相談してください — 重要なものは何でも検証してください([責任ある利用](/docs/security/responsible-use))。
- **音声とモバイル**で、どこにいても利用できます([音声モード](/docs/claude-app/voice-mode)、[モバイル](/docs/claude-app/mobile))。
- **あなたのデータはあなたのものです** — 自分が安心して共有できるものだけを共有してください([プライバシー](/docs/foundations/privacy))。
