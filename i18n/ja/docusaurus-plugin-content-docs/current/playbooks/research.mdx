AIは優れたリサーチの*加速装置*です — そして盲目的に信頼すれば危険なものでもあります。正しく使えば、何時間もの読書を数分に圧縮できます。不注意に使えば、自信たっぷりに情報源をでっち上げます。ここでは規律ある使い方を紹介します。

## 価値の高いリサーチの手法

- 長い文書や論文を要点と含意に**要約**する。
- 複数の情報源を表に**比較**する(何で一致し、何で対立しているか)。
- **ランドスケープを構築**する — あるトピックについての関係者、アプローチ、議論をマッピングする。
- 文書から特定の事実や数値を**抽出**する(引用付きで)。
- **プレッシャーテスト**する — 「これに対する最も強力な反論は何か?」

## グラウンディングの規律(交渉の余地なし)

```text
Using ONLY the text I provide, summarize the key findings and quote the exact
sentence supporting each one. If something isn't in the text, say "not stated."
Text: """{paste}"""
```

- **情報源を与える。** 重要なことについて記憶からの事実の想起を求めてはいけません — 資料を貼り付けるか、それを取得するツールを使ってください([RAG](/docs/foundations/rag))。
- **引用を要求する。** 「各主張を裏付ける文を引用して。」これにより、裏付けのない主張が可視化されます。
- **事実と推論を分離する。** 何が述べられているか対何が解釈されているかをラベル付けするよう求めてください。

:::warning でっち上げられた引用は典型的な落とし穴
モデルは本物そっくりの論文、著者、URLをでっち上げます。頼りにする前に**すべての引用が実在することを検証してください**。[ハルシネーション](/docs/foundations/hallucinations)を参照。
:::

## リサーチのフロー

1. 情報源を自分で集める(または取得ツールを使う)。
2. グラウンディングと引用を伴って、それぞれを要約する。
3. それらを横断してテーマ別の比較に統合する。
4. ギャップや対立、そして「だから何なのか」を特定する。
5. 重要な事実を原典と照合してスポットチェックする。
