**AILmanac 自体**への変更 — 構造、機能、主要なコンテンツの追加。（*Claude* 自体の変更については [今月の Claude](/docs/whats-new/this-month) を参照してください。）形式は [Keep a Changelog](https://keepachangelog.com/) に準拠しています。

## 未リリース

### 追加

- **Claude Code** および **Claude API** セクションの完全版（全ページ執筆済み）。
- **AI の基礎**、**プロンプティング**、**Claude.ai とアプリ**、**プレイブック**、**テンプレート**、**セキュリティ** の各セクションを完成。
- インタラクティブウィジェット: **Prompt Builder**、**Model Picker**、**Token Estimator**。
- ユーザータイプ別の **7 種類のスキルパック**（`/skills`）。
- ブランド化された **Open Graph** ソーシャルカード、プレミアムテーマ + アニメーション付きホームページ。
- 鮮度管理システム: 信頼できる唯一の情報源としての `data/models.json` + `<VerifyNote>` スタンプ。

### 変更

- プレミアムなビジュアルへのリデザイン（タイポグラフィ、配色、カスタムロゴ）。

## 補足

- これはコミュニティプロジェクトです — 貢献者はリポジトリで確認できます。
- あなたの変更をここに記録してほしいですか？ プルリクエストで言及してください（[貢献する](/docs/contribute/contribute-in-10-minutes)）。
- 変更を購読するには [GitHub リポジトリ](https://github.com/derob98/ailmanac) をウォッチしてください。
