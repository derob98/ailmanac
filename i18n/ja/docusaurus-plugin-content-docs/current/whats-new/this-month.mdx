Claude 全体で最近何が変わったかを平易な言葉でまとめたダイジェストです — すべての変更履歴を読まなくても「何が新しく、自分にとって何を意味するか」をざっと把握できます。新しいものから順に掲載しています。

:::info このページが最新に保たれる仕組み
エントリは、コミュニティと、上流の変更を検出して人間が執筆できるようフラグを立てる [自動ウォッチャー](/docs/contribute/automation) によって追加されます。各エントリには公式の情報源がリンクされています。足りないものがありますか？ [追加してください](/docs/contribute/contribute-in-10-minutes)。
:::

## エントリの読み方

各アップデートには **対象領域** のタグが付いています — 🗣️ アプリ（チャット/音声/モバイル）、🛠️ Claude Code、🔌 API、🧠 モデル — そして **「あなたにとって何を意味するか」** に答えます。

## 最近のアップデート

> _シードエントリ — 実際の、出典付きのアップデートがリリースされ次第、置き換えてください。_

- 🧠 **モデル** — 最新の Claude モデルラインアップと ID については [現行モデルと価格](/docs/whats-new/models-and-pricing) の表を確認してください。*意味すること:* 新しい構築は現行のデフォルトから始め、正確な ID はそこで調べましょう。
- 🛠️ **Claude Code** — 機能は頻繁にリリースされます（スキル、フック、MCP、スケジューリング）。*意味すること:* ページの `last verified` 日付が古そうに見える場合は、[公式の変更履歴](https://code.claude.com/docs/en/overview) と照合してください。
- 🔌 **API** — 構造化出力、拡張思考、プロンプトキャッシュなどの機能は進化しています。*意味すること:* 正確なパラメータに依存する前に、[API セクション](/docs/api/first-call) を再確認してください。
