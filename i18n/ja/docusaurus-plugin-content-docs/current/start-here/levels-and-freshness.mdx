2つの小さな慣習が、AILmanac をナビゲートしやすく、信頼しやすいものにしています。ここではその読み方を説明します。

## レベルバッジ

すべてのページは、誰向けかを示すバッジで始まります。

- <LevelBadge level="beginner" /> 予備知識を前提としません。初心者はここから。
- <LevelBadge level="intermediate" /> AIを少し使ったことがあり、さらに深く学びたい人向け。
- <LevelBadge level="advanced" /> エージェント、API、カスタム構成など、何かを構築している人向け。
- <LevelBadge level="all" /> どのレベルでも役立ちます。

バッジは**色だけでなくテキストに**依存しているため、誰にでも（スクリーンリーダーの利用者や色覚に違いのある読者を含めて）機能します。自分のレベルより上は飛ばしてもよいですし、準備ができたら背伸びして読んでもかまいません。[学習パス](/docs/start-here/learning-paths)が、読者タイプごとに理にかなった順序でページをつなげています。

## 鮮度: 「最終確認日」スタンプ

AIは速く動きます。変化するもの（モデル名、価格、上限、UIラベル、ベータ機能など）はすべて**変動しやすい事実**であり、AILmanac ではそれらに次のようなスタンプが付きます。

<VerifyNote lastVerified="2026-06-20" source="https://platform.claude.com/docs">
これが鮮度スタンプの見た目です。日付は人間が最後に確認した時点であり、リンクは常に優先される公式の情報源です。
</VerifyNote>

- **不変のページ**（「LLMとは」のような概念）には**日付が付きません** — それらは古びないので、日付はノイズになるだけです。
- **変動するページ**には日付が表示され、その詳細がどれくらい最新かがわかります。
- **古い日付はバグではなく、誠実さの表れです。** それは、その特定の事実を当然のものと思い込まずに再確認すべきだと教えてくれます。

## 唯一の確固たる保証

モデルの正体は**1か所**にあります。それは検証され日付が付けられた[現行モデルと価格](/docs/whats-new/models-and-pricing)の表です。モデルに関する事実は他の場所にハードコードされておらず、すべてがそこへリンクしています。だからこそ、信頼できる唯一の情報源が存在します。

## なぜこれが重要か

自信たっぷりに時代遅れなガイドは、ガイドがないよりも悪いものです。これらの慣習によって、私たちは何も変わらないふりをすることなく**網羅的**でいられます。そして*あなた*は、どの一文をどれだけ信頼すべきかを調整できます。
