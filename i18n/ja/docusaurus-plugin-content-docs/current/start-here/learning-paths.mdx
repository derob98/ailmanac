自分に最も近いと感じるトラックを選びましょう。それぞれが順序立てられたパスです。ステップは順番に進め、すでに知っていることは飛ばして構いません。所要時間はおおよその目安です。

<Callout type="objectives" items={["自分の目標に合う唯一のトラックを見つける — 初心者、ナレッジワーカー、ビルダー、チームリーダーのいずれか", "どこから始めるか推測するのではなく、明確で順序立てられた読み進め方をたどる", "各トラックにかかるおおよその時間を把握する（約45分〜約半日）", "どのトラックもしっくりこない場合にどこから始めるか"]} />

:::tip どのトラックが自分に合うか分からない？
**役職**ではなく**目標**で選びましょう。コーディングせずに成果を出したい？ 初心者。実際の仕事をもっと速く？ ナレッジワーカー。Claude Code や API をカスタマイズしたい？ ビルダー。AI を他の人たちに展開したい？ チームリーダー。
:::
