**AILmanac** は、**Claude — そしてあらゆるAI — を最大限に活用するための**、無料でコミュニティが作る**年鑑（almanac）**です。年鑑とは、繰り返し戻ってくる参考書のこと。網羅的で、平易で、常に最新に保たれたもの。それがここでの目標です。

## 誰のためのものか

意図的に、すべての人のためです。そして私たちは各ページのレベルをお伝えするので、迷ったり見下されたように感じたりすることはありません。

- <LevelBadge level="beginner" /> AIをほとんど使ったことがなく、確実な最初の成功体験が欲しい人。
- <LevelBadge level="intermediate" /> すでに Claude を使っていて、目に見えてより良い結果を出したい人。
- <LevelBadge level="advanced" /> Claude Code、API、エージェント、MCP で構築する人。

## 何が違うのか

すでに優れたリソースは存在します（[全体マップ](/docs/start-here/landscape-map)を参照）。AILmanac はリンク集**でも**、公式ドキュメントの複製**でもありません**。それは以下のようなものです。

- **意見がある** — まず*推奨される1つの方法*を示し、それから代替案を伝えます。
- **レベルタグ付き** — すべてのページにバッジが付いており、自分に合ったものを読めます。
- **テンプレートが豊富** — [コピペで使えるスターター](/docs/templates)を30秒で使えます。
- **鮮度について正直** — 変化する事実（モデル名、価格、機能）には「最終確認日」と情報源へのリンクが付きます。

## これが「ではない」もの

:::warning 公式ドキュメントが信頼できる情報源です
AILmanac と[公式 Anthropic ドキュメント](https://platform.claude.com/docs)が食い違う場合は、**公式ドキュメントが優先されます**。AIツールは速く変化します。私たちは最新を保つよう努めていますが、重要なことは必ずリンク先の情報源で確認してください。
:::

また、私たちはあなたのデータを保存せず、あなたのアカウントを見ることもできず、重大な判断をあなたの代わりに下すこともありません — [責任ある利用](/docs/security)を参照。
