**AILmanac** は、**Claude を — そしてあらゆる AI を — 最大限に活用する**ための、無料でコミュニティが作る**年鑑（almanac）**です。年鑑とは、何度も立ち返る参考書のこと。網羅的で、平易な言葉で書かれ、常に最新に保たれています。それがここでの目標です。

<Callout type="objectives" items={["AILmanac とは何か — そして何をあえて目指さないのかを理解する", "自分に合ったレベルを見つける: 初級・中級・上級", "最速の入り口を選び、1 分以内に読み始める"]} />

## 誰のためのものか

意図的に、すべての人のためです — そして各ページのレベルを明示しているので、迷ったり見下されたりすることはありません:

| レベル | こんなあなたに |
| --- | --- |
| <LevelBadge level="beginner" /> | AI をほとんど使ったことがなく、確実な最初の成功体験がほしい。 |
| <LevelBadge level="intermediate" /> | すでに Claude を使っていて、目に見えて良い結果がほしい。 |
| <LevelBadge level="advanced" /> | Claude Code、API、エージェント、MCP で構築している。 |

## 何が違うのか

すでに優れたリソースは存在します（[ランドスケープマップ](/docs/start-here/landscape-map)を参照）。AILmanac は**リンク集**でも、公式ドキュメントの**コピー**でもありません。これは:

- **意見を持つ** — まず*推奨される一つの方法*を伝え、それから代替案を示します。
- **レベルタグ付き** — すべてのページにバッジが付き、自分に合った内容を読めます。
- **テンプレート豊富** — 30 秒で使える[コピペ用スターター](/docs/templates)。
- **鮮度に正直** — 変わりうる事実（モデル名、価格、機能）には「最終確認日」と出典へのリンクが付きます。

## これは何「ではない」のか

:::warning 公式ドキュメントが信頼できる情報源です
AILmanac と[公式 Anthropic ドキュメント](https://platform.claude.com/docs)が食い違う場合、**公式ドキュメントが優先**されます。AI ツールは急速に変化します。私たちは最新に保つよう努めていますが、重要なことは必ずリンク先の出典で確認してください。
:::

また、私たちはあなたのデータを保存せず、アカウントを見ることもできず、重要な判断を代わりに下すこともありません — [責任ある利用](/docs/security)を参照してください。
