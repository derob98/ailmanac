AILmanac は真空の中に存在しているわけではなく、すべてを置き換えるふりもしません。ここでは最良のリソースを正直にマップ化します。そして、それらを私たちの**代わりに**いつ使うべきかを示します。

## 信頼できる情報源（source of truth）

- **[Anthropic ドキュメント](https://docs.anthropic.com)** — Claude、API、モデル、機能に関する権威ある公式リファレンス。**AILmanac と公式ドキュメントが食い違う場合は、公式ドキュメントが優先されます。** 正確で最新の詳細にはドキュメントを、意見のある層分けされた実践的ガイダンスには私たちを使ってください。

## Anthropic 公式リソース

- **[Prompt Library](https://docs.anthropic.com/en/resources/prompt-library/library)** — すぐ使える例示プロンプト集。
- **[Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)** — 実行可能なAPIレシピ（ツール利用、RAG、ビジョン）。
- **[Claude Code ドキュメント](https://code.claude.com/docs/en/overview)** — CLI/IDE のリファレンス。
- **[Anthropic Academy / Learn](https://www.anthropic.com/learn)** — 無料のガイド付きコース。
- **[Model Context Protocol](https://modelcontextprotocol.io)** — オープンな MCP 標準＋サーバーディレクトリ。

## コミュニティ・一般

- **[awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code)** — 厳選されたコマンド、フック、ワークフロー集。
- **[Prompt Engineering Guide](https://www.promptingguide.ai)** — 深く、ベンダー中立なプロンプティングのリファレンス。
- **[Learn Prompting](https://learnprompting.org)** — 体系的で初心者にやさしいプロンプティングコース。

## いつ何を使うか

| やりたいこと | 行き先 |
|---|---|
| 正確で最新のAPI/機能の詳細 | 公式ドキュメント |
| コピペで使える例示プロンプト | Prompt Library / 私たちの[テンプレート](/docs/templates/prompts) |
| 実行可能なAPIコード | Cookbook |
| 意見のある、レベルタグ付きの道筋 | **AILmanac** |
| プロンプティングの深い理論リファレンス | promptingguide.ai |
| ガイド付きコース | Anthropic Academy |

## AILmanac の位置づけ

私たちは**意見のあるフィールドガイド**です。まず推奨される1つの方法を示し、レベルでタグ付けし、テンプレートが豊富で、鮮度について正直で、Claude **と**一般的なAIの概念の両方をカバーします。私たちのことは、適切なタイミングで適切な深掘りリソースを指し示す地図だと考えてください。そして、それらすべてが意味を持つように基礎を教えます。
