<VerifyNote lastVerified="2026-06-20" source="https://docs.anthropic.com/en/docs/about-claude/models">
Графики устаревания и детали миграций берутся от Anthropic и меняются — всегда сверяйте даты и замены в официальной документации перед миграцией.
</VerifyNote>

Модели снимаются с поддержки, параметры переименовываются, поведение меняется. Эта страница отслеживает **ломающие изменения**, чтобы обновления не застали вас врасплох. Если вы строите что-то на API, просматривайте её время от времени.

## Почему это важно

Закреплённый ID модели рано или поздно будет **признан устаревшим**, а затем **удалён**. Код, в который этот ID жёстко зашит, перестанет работать. Немного гигиены избавит вас от сюрпризов в 2 часа ночи.

## Правила выживания

- **Считывайте ID моделей из конфигурации**, никогда не разбрасывайте литералы по всей кодовой базе — тогда миграция станет правкой в одну строку.
- **Следите за анонсами** дат устаревания и рекомендуемых замен.
- **Перезапускайте свои [оценки (evals)](/docs/foundations/evals)** при смене модели — поведение может измениться даже на «более новой и лучшей» модели.
- **Тестируйте до дедлайна**, а не после удаления.

## Типичные виды изменений

| Изменение | Пример | Что делать |
|---|---|---|
| Модель снята с поддержки | Старый ID модели перестаёт работать | Перейдите на рекомендуемого преемника; перезапустите оценки |
| Параметр заменён | Ручка reasoning/budget заменена на `effort` | Обновите вызовы; см. [Thinking & Effort](/docs/api/thinking-and-effort) |
| Изменение значения по умолчанию | Новая модель/поведение по умолчанию | Закрепляйте осознанно; проверьте, что ваши промпты по-прежнему работают |
| Обновление эндпоинта/SDK | Новая мажорная версия SDK | Прочитайте руководство по миграции; обновляйтесь обдуманно |

## Текущее отслеживание

> _Затравка — замените реальными уведомлениями об устаревании со ссылками на источники по мере их анонсирования._

- Сверьте свои ID моделей с таблицей [Текущие модели и цены](/docs/whats-new/models-and-pricing); если вы используете старый ID, запланируйте миграцию.
- Других ломающих изменений здесь пока не зафиксировано. Отслеживайте [официальную документацию по моделям](https://docs.anthropic.com/en/docs/about-claude/models) — это авторитетный график.
