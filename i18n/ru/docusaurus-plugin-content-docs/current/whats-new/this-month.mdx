## Как читать запись

Каждое обновление помечено по **поверхности** — 🗣️ Приложения (чат/голос/мобильные), 🛠️ Claude Code, 🔌 API, 🧠 Модели — и отвечает на вопрос **«что это значит для вас»**.

## Недавние обновления

> _Затравочная запись — замените реальными обновлениями со ссылками на источники по мере их выхода._

- 🧠 **Модели** — Смотрите таблицу [Текущие модели и цены](/docs/whats-new/models-and-pricing), чтобы узнать актуальный набор моделей Claude и их ID. *Что это значит:* начинайте новые проекты на текущей модели по умолчанию и ищите точные ID там.
- 🛠️ **Claude Code** — Функции выходят часто (навыки, хуки, MCP, планирование). *Что это значит:* если дата `last verified` на странице выглядит устаревшей, сверьтесь с [официальным журналом изменений](https://docs.anthropic.com/en/docs/claude-code/overview).
- 🔌 **API** — Возможности вроде структурированного вывода, расширенного мышления и кэширования промптов развиваются. *Что это значит:* перепроверьте [раздел API](/docs/api/first-call), прежде чем полагаться на конкретные параметры.
