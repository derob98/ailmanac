Набор инструментов из многоразовых паттернов промптинга. В каждой карточке: что это, когда к ней обращаться и шаблон для копирования. Они складываются — комбинируйте свободно. Здесь впервые? Сначала начните с [Основ промптинга](/docs/prompting/basics).

## Zero-shot (просто спросите, чётко)
**Когда:** простые, распространённые задачи. **Почему:** наименьшие усилия, зачастую достаточно.
```text
Summarize the text below in 5 bullets for a busy manager.
TEXT: """{paste}"""
```

## Few-shot (покажите примеры)
**Когда:** вам нужен конкретный формат или стиль. **Почему:** примеры обучают быстрее, чем описания. См. [Few-shot как надо](/docs/prompting/few-shot).
```text
Classify the sentiment as positive/negative/neutral.
Review: "Loved it, shipped fast" → positive
Review: "Broke on day one" → negative
Review: "{new review}" →
```

## Цепочка рассуждений (думай шаг за шагом)
**Когда:** математика, логика, многошаговые рассуждения. **Почему:** рассуждение перед ответом повышает точность.
```text
Work through this step by step, then give the final answer on its own line.
PROBLEM: {problem}
```

## Декомпозиция (разбейте на части)
**Когда:** большие, размытые задачи. **Почему:** меньшие подзадачи проще и надёжнее.
```text
First, list the sub-tasks needed to {goal}. Then do them one at a time,
showing each result before moving on.
```

## Ролевой промптинг (назначьте персону)
**Когда:** вам нужна конкретная оптика или стандарт. **Почему:** настраивает на релевантное поведение.
```text
You are a meticulous senior copy editor. Edit the text for clarity and concision
without changing meaning. Show a bullet list of the changes you made.
```

## Ограничение формата вывода
**Когда:** результат подаётся в другой инструмент, или вам нужна согласованность. **Почему:** предсказуемые формы пригодны к использованию.
```text
Reply with ONLY valid JSON matching: {"title": string, "tags": string[]}.
No prose, no markdown fences.
```

## Самосогласованность / самокритика
**Когда:** важна корректность. **Почему:** второй проход ловит ошибки первого прохода.
```text
Answer the question. Then, in a section called "Check", critique your own answer
for errors or missing cases and revise if needed.
```

## Мета-промптинг (пусть ИИ улучшит ваш промпт)
**Когда:** ваш промпт работает слабо. **Почему:** модель хорошо подмечает, чего ей не хватает.
```text
Here's my prompt: "{prompt}". Ask me the 3 questions that would most improve it,
then rewrite it incorporating sensible defaults.
```

## Итеративная доработка
**Когда:** почти всегда. **Почему:** первый вывод — черновик, а не пункт назначения.
```text
{after a result} Shorter and more concrete. Keep the second paragraph. Add one
example. Drop the marketing tone.
```

:::tip Комбинируйте их
Сильный реальный промпт часто такой: **роль** + **пример** + **ограничение вывода** + разрешение **сказать «я не знаю».** Наслаивайте карточки.
:::
