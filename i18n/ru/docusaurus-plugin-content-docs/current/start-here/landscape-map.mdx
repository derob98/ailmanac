AILmanac не существует в вакууме, и мы не будем притворяться, что заменяем всё. Вот честная карта лучших ресурсов — и того, когда использовать их **вместо** нас.

## Источник истины

- **[Документация Anthropic](https://docs.anthropic.com)** — авторитетный справочник по Claude, API, моделям и возможностям. **Когда AILmanac и документация расходятся, побеждает документация.** Используйте её для точных, актуальных деталей; используйте нас для аргументированного, послойного руководства по принципу «учимся на практике».

## Официальные ресурсы Anthropic

- **[Библиотека промптов](https://docs.anthropic.com/en/resources/prompt-library/library)** — готовые примеры промптов.
- **[Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)** — исполняемые рецепты для API (использование инструментов, RAG, зрение).
- **[Документация Claude Code](https://code.claude.com/docs/en/overview)** — справочник по CLI/IDE.
- **[Anthropic Academy / Learn](https://www.anthropic.com/learn)** — бесплатные курсы с сопровождением.
- **[Model Context Protocol](https://modelcontextprotocol.io)** — открытый стандарт MCP + каталог серверов.

## Сообщество и общее

- **[awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code)** — отобранные команды, хуки и рабочие процессы.
- **[Prompt Engineering Guide](https://www.promptingguide.ai)** — глубокий, вендор-нейтральный справочник по промптингу.
- **[Learn Prompting](https://learnprompting.org)** — структурированный, дружелюбный к новичкам курс по промптингу.

## Когда что использовать

| Вы хотите… | Перейдите к |
|---|---|
| Точные актуальные детали API/возможностей | Официальная документация |
| Готовые примеры промптов для копирования | Библиотека промптов / наши [шаблоны](/docs/templates/prompts) |
| Исполняемый код для API | Cookbook |
| Аргументированный путь с метками уровней | **AILmanac** |
| Глубокий теоретический справочник по промптингу | promptingguide.ai |
| Курс с сопровождением | Anthropic Academy |

## Где место AILmanac

Мы — **аргументированный полевой справочник**: сначала один рекомендуемый способ, с метками по уровням, богатый шаблонами, честный об актуальности и охватывающий как Claude, **так и** общие концепции ИИ. Считайте нас картой, которая указывает на нужный глубокий ресурс в нужное время — и обучает основам, чтобы все они обрели смысл.
