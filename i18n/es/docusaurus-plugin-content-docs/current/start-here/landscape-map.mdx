AILmanac no existe en el vacío, y no vamos a fingir que lo reemplaza todo. Aquí tienes un mapa honesto de los mejores recursos, y cuándo usarlos **en lugar de** nosotros.

## La fuente de verdad

- **[Documentación de Anthropic](https://docs.anthropic.com)**: la referencia autoritativa para Claude, la API, los modelos y las funciones. **Cuando AILmanac y la documentación discrepen, gana la documentación.** Úsala para detalles exactos y actuales; úsanos a nosotros para una guía opinada, por niveles y de aprender haciendo.

## Recursos oficiales de Anthropic

- **[Biblioteca de prompts](https://docs.anthropic.com/en/resources/prompt-library/library)**: prompts de ejemplo listos para usar.
- **[Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)**: recetas ejecutables de la API (uso de herramientas, RAG, visión).
- **[Documentación de Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview)**: la referencia de la CLI/IDE.
- **[Anthropic Academy / Learn](https://www.anthropic.com/learn)**: cursos guiados gratuitos.
- **[Model Context Protocol](https://modelcontextprotocol.io)**: el estándar abierto MCP + directorio de servidores.

## Comunidad y general

- **[awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code)**: comandos, hooks y flujos de trabajo seleccionados.
- **[Prompt Engineering Guide](https://www.promptingguide.ai)**: referencia profunda y neutral en cuanto a proveedor sobre prompting.
- **[Learn Prompting](https://learnprompting.org)**: curso de prompting estructurado y apto para principiantes.

## Cuándo usar qué

| Quieres… | Ve a |
|---|---|
| Detalles exactos y actuales de la API/funciones | Documentación oficial |
| Prompts de ejemplo para copiar y pegar | Biblioteca de prompts / nuestras [plantillas](/docs/templates/prompts) |
| Código de API ejecutable | Cookbook |
| Un camino opinado y etiquetado por nivel | **AILmanac** |
| Una referencia teórica profunda sobre prompting | promptingguide.ai |
| Un curso guiado | Anthropic Academy |

## Dónde encaja AILmanac

Somos la **guía de campo opinada**: primero una forma recomendada, etiquetada por nivel, rica en plantillas, honesta sobre la actualidad y que abarca Claude **y** conceptos generales de IA. Piensa en nosotros como el mapa que te señala el recurso profundo adecuado en el momento adecuado, y que enseña los fundamentos para que todos cobren sentido.
