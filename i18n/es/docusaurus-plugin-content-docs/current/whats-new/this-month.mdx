Un resumen en lenguaje claro de lo que ha cambiado recientemente en Claude — para que puedas ojear "qué hay de nuevo y qué significa para mí" sin leer todos los registros de cambios. Lo más reciente primero.

:::info Cómo se mantiene actualizada esta página
Las entradas las añaden la comunidad y un [vigilante automatizado](/docs/contribute/automation) que detecta cambios upstream para que una persona los redacte. Cada entrada enlaza su fuente oficial. ¿Echas algo en falta? [Añádelo](/docs/contribute/contribute-in-10-minutes).
:::

## Cómo leer una entrada

Cada actualización está etiquetada por **superficie** — 🗣️ Aplicaciones (chat/voz/móvil), 🛠️ Claude Code, 🔌 API, 🧠 Modelos — y responde a **"qué significa para ti."**

## Actualizaciones recientes

> _Entrada de muestra — sustitúyela por actualizaciones reales y con fuente a medida que se publiquen._

- 🧠 **Modelos** — Consulta la tabla [Modelos y precios actuales](/docs/whats-new/models-and-pricing) para ver la última lista de modelos de Claude y sus IDs. *Qué significa:* empieza los nuevos desarrollos con el modelo predeterminado actual y busca allí los IDs exactos.
- 🛠️ **Claude Code** — Las funcionalidades se publican con frecuencia (skills, hooks, MCP, programación de tareas). *Qué significa:* si la fecha de `última verificación` de una página parece antigua, contrástala con el [registro de cambios oficial](https://code.claude.com/docs/en/overview).
- 🔌 **API** — Capacidades como la salida estructurada, el pensamiento extendido y el almacenamiento en caché de prompts evolucionan. *Qué significa:* vuelve a comprobar la [sección de la API](/docs/api/first-call) antes de depender de parámetros exactos.
