## Sin publicar

### Añadido
- Secciones completas de **Claude Code** y **Claude API** (todas las páginas escritas).
- Secciones de **Fundamentos de IA**, **Prompting**, **Claude.ai y aplicaciones**, **Playbooks**, **Plantillas** y **Seguridad** completadas.
- Widgets interactivos: **Prompt Builder**, **Model Picker**, **Token Estimator**.
- **7 packs de skills** (`/skills`) para distintos tipos de usuarios.
- Tarjeta social **Open Graph** con marca; tema premium + página de inicio animada.
- Sistema de actualización: `data/models.json` como fuente de verdad + sellos `<VerifyNote>`.
