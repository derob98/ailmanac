La API permite que *tu* programa hable con Claude. Tres pasos: consigue una clave, instala el SDK y envía un mensaje.

## 1. Consigue una clave de API

Crea una en la [Consola de Anthropic](https://console.anthropic.com). Después, configúrala como variable de entorno para que nunca quede en tu código:

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

:::warning Nunca subas tu clave al control de versiones
Mantén las claves en variables de entorno o en un gestor de secretos, nunca en el control de versiones. Consulta [Seguridad](/docs/security).
:::

## 3. Haz la llamada

Cada solicitud es una lista de `messages`. El modelo responde con `content`.

<Tabs groupId="lang">
<TabItem value="python" label="Python">

```python
import anthropic

client = anthropic.Anthropic()  # reads ANTHROPIC_API_KEY from the environment

message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "In one sentence, what is an API?"}
    ],
)

print(message.content[0].text)
```

</TabItem>
<TabItem value="ts" label="TypeScript">

```ts
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic(); // reads ANTHROPIC_API_KEY from the environment

const message = await client.messages.create({
  model: "claude-sonnet-4-6",
  max_tokens: 1024,
  messages: [
    { role: "user", content: "In one sentence, what is an API?" },
  ],
});

console.log(message.content[0].text);
```

</TabItem>
<TabItem value="curl" label="cURL">

```bash
curl https://api.anthropic.com/v1/messages \
  --header "x-api-key: $ANTHROPIC_API_KEY" \
  --header "anthropic-version: 2023-06-01" \
  --header "content-type: application/json" \
  --data '{
    "model": "claude-sonnet-4-6",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "In one sentence, what is an API?"}
    ]
  }'
```

</TabItem>
</Tabs>

## Qué acaba de pasar

- **`model`** — qué Claude usar. No lo codifiques a ciegas; consulta [Elegir un modelo](/docs/api/choosing-a-model).
- **`max_tokens`** — un tope para la longitud de la respuesta (en [tokens](/docs/foundations)). *No* define la ventana de contexto.
- **`messages`** — la conversación hasta el momento. La API es **sin estado**: para continuar un chat, vuelve a enviar todo el historial cada vez.
- **`system`** (opcional) — una instrucción de nivel superior que establece el rol de Claude para la llamada.
