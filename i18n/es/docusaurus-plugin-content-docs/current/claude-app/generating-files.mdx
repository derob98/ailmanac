Más allá del texto en pantalla, Claude puede producir **archivos reales y descargables** — documentos de Word, presentaciones de PowerPoint, hojas de cálculo de Excel, PDF — con formato y listos para usar. Ideal cuando necesitas un entregable, no solo una respuesta.

## Qué puedes pedir

- **Word (.docx)** — informes, cartas, propuestas, documentos con formato.
- **PowerPoint (.pptx)** — presentaciones a partir de un esquema o notas.
- **Excel (.xlsx)** — hojas de cálculo con datos, fórmulas, modelos sencillos.
- **PDF** — versiones listas para imprimir de lo anterior.

## Cómo obtener un buen archivo

1. **Describe el entregable**: tipo, audiencia, extensión, secciones, tono.
2. **Aporta el contenido o la fuente** (notas, datos, un esquema) — o pide a Claude que lo redacte primero.
3. **Pide el archivo** ("exporta esto como .docx"), luego **itera** ("añade una portada", "convierte la tabla en un gráfico", "ajusta la diapositiva 3").

```text
Turn these notes into a 6-slide investor update deck (.pptx): traction, revenue,
roadmap, team, ask. Clean and minimal. Notes: {paste}
```

## Consejos

- **Aporta estructura.** Encabezados/secciones en tu petición → una salida más limpia.
- **Aporta datos reales** para las hojas de cálculo — y **verifica** cualquier número calculado ([Alucinaciones](/docs/foundations/hallucinations)); no entregues cifras que no hayas comprobado.
- **Refina antes de exportar** — es más fácil corregir en el chat que en el archivo final.
- Estas capacidades funcionan gracias a las [Skills](/docs/claude-code/skills) por debajo — la misma idea que tú mismo puedes crear.
