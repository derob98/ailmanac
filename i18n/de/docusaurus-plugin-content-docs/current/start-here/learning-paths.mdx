Wähle den Track, der am ehesten zu dir passt. Jeder ist ein geordneter Pfad — arbeite sie der Reihe nach ab und überspringe, was du schon kennst. Die Zeitangaben sind grobe Richtwerte.

## 🌱 Der neugierige Neuling · ~45 Min.

Du möchtest KI gut nutzen, ohne Programmierer zu werden.

1. [Deine ersten 5 Minuten](/docs/start-here/your-first-5-minutes) <LevelBadge level="beginner" />
2. [Welches Claude sollte ich verwenden?](/docs/start-here/which-claude) <LevelBadge level="beginner" />
3. [Grundlagen des Promptings](/docs/prompting/basics) <LevelBadge level="beginner" />
4. [Was ist ein LLM?](/docs/foundations) <LevelBadge level="beginner" />
5. [Alltägliche Produktivität](/docs/playbooks/productivity) <LevelBadge level="beginner" />

## 💼 Der Wissensarbeiter · ~2 Std.

Du möchtest, dass KI deinen echten Job schneller macht — Schreiben, Recherche, Analyse.

1. Der Track des neugierigen Neulings (oben), dann…
2. [Bibliothek der Prompt-Muster](/docs/prompting) <LevelBadge level="all" />
3. [Halluzinationen & wie man sie reduziert](/docs/foundations) <LevelBadge level="intermediate" />
4. Dein Playbook: [Schreiben](/docs/playbooks), [Recherche](/docs/playbooks) oder [Datenanalyse](/docs/playbooks)
5. [Projects: dauerhafte Arbeitsbereiche](/docs/claude-app) und [Wiederverwendbare Prompt-Vorlagen](/docs/templates)

## 🛠️ Der Entwickler / Builder · ~½ Tag

Du schreibst Code und möchtest Claude Code anpassen und auf der API aufbauen.

1. [Was Claude Code ist](/docs/claude-code/what-is-claude-code) <LevelBadge level="beginner" />
2. [CLAUDE.md & Memory-Dateien](/docs/claude-code) <LevelBadge level="beginner" /> + [Plan Mode](/docs/claude-code)
3. [Berechtigungen](/docs/claude-code) und [settings.json](/docs/claude-code) <LevelBadge level="intermediate" />
4. [Dein erster API-Aufruf](/docs/api/first-call) → [Tool Use](/docs/api) → [Agenten bauen](/docs/api) <LevelBadge level="advanced" />
5. [Walkthrough: Claude Code für ein echtes Repo anpassen](/docs/walkthroughs) <LevelBadge level="intermediate" />
6. [Sicherheit & verantwortungsvolle Nutzung](/docs/security) <LevelBadge level="advanced" />

## 👥 Der Team-Lead · ~3 Std.

Du führst KI in einem Team ein und legst Wert auf Konsistenz, Sicherheit und Kosten.

1. Die Grundlagen des Builder-Tracks (1–3 oben)
2. [Plugins & Marketplaces](/docs/claude-code) — ein gemeinsames Toolkit ausliefern <LevelBadge level="advanced" />
3. [Abwägungen zwischen Kosten & Latenz](/docs/foundations) und [Ein Modell wählen](/docs/api/choosing-a-model) <LevelBadge level="intermediate" />
4. [Agenten absichern](/docs/security) und [Autonome Läufe härten](/docs/security) <LevelBadge level="advanced" />
5. [Verantwortungsvolle Nutzung & Verifikation](/docs/security) — Team-Normen festlegen <LevelBadge level="all" />
