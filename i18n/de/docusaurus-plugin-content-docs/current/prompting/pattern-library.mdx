Ein Werkzeugkasten wiederverwendbarer Prompting-Patterns. Jede Karte: was es ist, wann man darauf zurückgreift, und eine Copy-and-paste-Vorlage. Sie lassen sich stapeln — kombiniere sie nach Belieben. Neu hier? Beginne zuerst mit den [Grundlagen des Promptings](/docs/prompting/basics).

## Zero-Shot (einfach klar fragen)
**Wann:** einfache, gängige Aufgaben. **Warum:** geringster Aufwand, oft ausreichend.
```text
Summarize the text below in 5 bullets for a busy manager.
TEXT: """{paste}"""
```

## Few-Shot (Beispiele zeigen)
**Wann:** du brauchst ein bestimmtes Format oder einen bestimmten Stil. **Warum:** Beispiele lehren schneller als Beschreibungen. Siehe [Few-Shot richtig gemacht](/docs/prompting/few-shot).
```text
Classify the sentiment as positive/negative/neutral.
Review: "Loved it, shipped fast" → positive
Review: "Broke on day one" → negative
Review: "{new review}" →
```

## Chain-of-Thought (Schritt für Schritt denken)
**Wann:** Mathematik, Logik, mehrstufiges Schlussfolgern. **Warum:** Nachdenken vor dem Antworten verbessert die Genauigkeit.
```text
Work through this step by step, then give the final answer on its own line.
PROBLEM: {problem}
```

## Zerlegung (in Teile aufgliedern)
**Wann:** große, unscharfe Aufgaben. **Warum:** kleinere Teilaufgaben sind einfacher und zuverlässiger.
```text
First, list the sub-tasks needed to {goal}. Then do them one at a time,
showing each result before moving on.
```

## Rollen-Prompting (eine Persona zuweisen)
**Wann:** du willst eine bestimmte Perspektive oder einen bestimmten Maßstab. **Warum:** stimmt relevantes Verhalten ein.
```text
You are a meticulous senior copy editor. Edit the text for clarity and concision
without changing meaning. Show a bullet list of the changes you made.
```

## Ausgabeformat-Vorgabe
**Wann:** das Ergebnis fließt in ein anderes Tool, oder du willst Konsistenz. **Warum:** vorhersehbare Formen sind nutzbar.
```text
Reply with ONLY valid JSON matching: {"title": string, "tags": string[]}.
No prose, no markdown fences.
```

## Selbstkonsistenz / Selbstkritik
**Wann:** Korrektheit ist wichtig. **Warum:** ein zweiter Durchgang fängt Fehler des ersten Durchgangs ab.
```text
Answer the question. Then, in a section called "Check", critique your own answer
for errors or missing cases and revise if needed.
```

## Meta-Prompting (lass die KI deinen Prompt verbessern)
**Wann:** dein Prompt liefert zu schwache Ergebnisse. **Warum:** das Modell erkennt gut, was ihm fehlt.
```text
Here's my prompt: "{prompt}". Ask me the 3 questions that would most improve it,
then rewrite it incorporating sensible defaults.
```

## Iterative Verfeinerung
**Wann:** fast immer. **Warum:** die erste Ausgabe ist ein Entwurf, nicht das Ziel.
```text
{after a result} Shorter and more concrete. Keep the second paragraph. Add one
example. Drop the marketing tone.
```

:::tip Kombiniere sie
Ein starker realer Prompt ist oft: eine **Rolle** + ein **Beispiel** + eine **Ausgabevorgabe** + die Erlaubnis, **„Ich weiß es nicht" zu sagen.** Schichte die Karten übereinander.
:::
