## Wie man einen Eintrag liest

Jedes Update ist nach **Oberfläche** getaggt — 🗣️ Apps (Chat/Sprache/Mobil), 🛠️ Claude Code, 🔌 API, 🧠 Modelle — und beantwortet **„was es für dich bedeutet."**

## Aktuelle Updates

> _Platzhalter-Eintrag — ersetze ihn durch echte, belegte Updates, sobald sie ausgeliefert werden._

- 🧠 **Modelle** — Prüfe die Tabelle [Aktuelle Modelle & Preise](/docs/whats-new/models-and-pricing) für die neueste Claude-Modellpalette und IDs. *Was es bedeutet:* Beginne neue Projekte mit dem aktuellen Default und schlage die genauen IDs dort nach.
- 🛠️ **Claude Code** — Funktionen werden häufig ausgeliefert (Skills, Hooks, MCP, Scheduling). *Was es bedeutet:* Wenn das `last verified`-Datum einer Seite alt aussieht, gleiche es mit dem [offiziellen Changelog](https://code.claude.com/docs/en/overview) ab.
- 🔌 **API** — Fähigkeiten wie strukturierte Ausgabe, Extended Thinking und Prompt Caching entwickeln sich weiter. *Was es bedeutet:* Überprüfe den [API-Abschnitt](/docs/api/first-call) erneut, bevor du dich auf genaue Parameter verlässt.
