Vision, PDF et fichiers en entrée

🟡Intermédiaire
✅
Dernière vérification : 2026-06-20.

Les types de fichiers pris en charge, les limites de taille et la surface de la Files API changent — confirmez dans la documentation officielle vision/PDF.

Vérifiez auprès de la source officielle.
🎯What you'll learn
  • ✅Pourquoi Claude est multimodal : images et documents dans un même message
  • ✅Les trois façons d'envoyer un fichier : Base64, URL et la Files API
  • ✅Quand réutiliser un téléversement par file_id pour économiser coût et temps
  • ✅Ce pour quoi la vision excelle — extraction, compréhension visuelle, questions-réponses sur documents
  • ✅Comment l'utiliser en toute sécurité : limites de résolution et vérification des chiffres critiques

Claude est multimodal : vous pouvez inclure des images et des documents (comme des PDF) dans un message et poser des questions à leur sujet — extraire des données, décrire un graphique, résumer un contrat, lire une capture d'écran.

Trois façons d'envoyer un fichier​

Guided walkthrough1 of 3
  1. Encodez les octets en ligne dans la requête. Le plus simple pour les cas ponctuels.
  2. Pointez vers un fichier hébergé.
  3. Téléversez une fois, puis référencez-le par file_id à travers de nombreuses requêtes (évite de re-téléverser de gros fichiers).

Voici à quoi ressemble un bloc de contenu image aux côtés d'une question textuelle :

# Conceptual: an image content block alongside text (see docs for exact fields)
messages=[{
  "role": "user",
  "content": [
    {"type": "image", "source": {"type": "base64", "media_type": "image/png", "data": b64}},
    {"type": "text", "text": "What does this chart show? Give the top 3 takeaways."},
  ],
}]

Les PDF fonctionnent de façon similaire (un bloc de contenu document) ; Claude peut lire le texte et la mise en page visuelle, et vous pouvez demander des citations renvoyant à l'endroit du document d'où provient une réponse.

Ce pour quoi elle excelle​

✨Pro tip
  • →Extraction — extraire des données structurées de factures, formulaires, tableaux (à associer à la Sortie structurée : /docs/api/structured-output).
  • →Compréhension visuelle — graphiques, diagrammes, captures d'écran, interfaces.
  • →Questions-réponses sur documents — poser des questions à travers un long PDF avec citations.

Astuces​

💡Key takeaways
  • ⭐Réutilisez avec file_id quand vous envoyez le même gros document de façon répétée — moins cher et plus rapide.
  • ⭐La résolution/taille comptent — les très grandes images peuvent être réduites ; vérifiez les limites.
  • ⭐Vérifiez les chiffres extraits — la vision est puissante mais pas infaillible ; contrôlez les chiffres critiques.
⚠️Watch out
  • •La vision est puissante mais pas infaillible — vérifiez toujours les chiffres critiques. Voir Hallucinations : /docs/foundations/hallucinations.
