هذا الشهر في Claude

جميع المستويات
آخر تحقّق: 2026-06-24.

هذه الصفحة ملخص متجدد يُنسَّق يدوياً. للاطلاع على القائمة الكاملة والموثوقة بالتغييرات، راجع دائماً وثائق Anthropic الرسمية وسجل تغييرات Claude Code.

تأكّد من المصدر الرسمي.

ملخص بلغة واضحة لما تغيّر مؤخراً عبر Claude — حتى تتمكن من تصفح "ما الجديد وماذا يعني لي" دون قراءة كل سجل تغييرات. الأحدث أولاً.

:::info كيف تبقى هذه الصفحة محدَّثة تُضاف الإدخالات من قِبل المجتمع ومن مراقب آلي يُشير إلى التغييرات الأولية ليكتب عنها إنسان. كل إدخال يربط مصدره الرسمي. لاحظت شيئاً مفقوداً؟ أضِفه. :::

كيف تقرأ إدخالاً​

كل تحديث موسوم بحسب السطح — 🗣️ التطبيقات (الدردشة/الصوت/الجوال)، 🛠️ Claude Code، 🔌 واجهة الـ API، 🧠 النماذج — ويجيب عن "ماذا يعني لك."

أحدث التحديثات​

  • 🧠 النماذج — أُطلق Claude Sonnet 5 في 30 يونيو 2026 وهو الآن النموذج الافتراضي في Claude Code، بنافذة سياق أصلية بحجم 1M-token وتسعير تمهيدي حتى August 31, 2026. إنه فئة Sonnet المتوازنة الجديدة "ابدأ من هنا"، ليحلّ محل Claude Sonnet 4.6 (وهو الآن نموذج قديم). ماذا يعني هذا: إذا كنت تثبّت نموذجاً، فانتقل إلى claude-sonnet-5؛ وتحقق دائماً من جدول النماذج والأسعار الحالية للاطلاع على المعرّفات الدقيقة والأسعار الحيّة قبل الاعتماد على رقم من تدوينة.
    آخر تحقّق: 2026-07-01. معرّف الـ API هو claude-sonnet-5؛ فئة Sonnet؛ سياق 1M-token؛ حد أقصى للمخرجات 128k؛ حد المعرفة يناير 2026. يُطبَّق التسعير التمهيدي البالغ $2 / $10 لكل MTok حتى Aug 31, 2026 (القياسي $3 / $15). أصبح الآن الافتراضي في Claude Code وفقاً لـسجل التغييرات (v2.1.197، June 30 2026). تأكّد من المصدر الرسمي.
  • 🛠️ Claude Code — سير العمل الديناميكي متاح في معاينة بحثية: اطلب واحداً (أو اضبط /effort ultracode) وسيكتب Claude سكربت JavaScript يُنسّق عشرات إلى مئات من الوكلاء الفرعيين في الخلفية — لعمليات التدقيق على مستوى قاعدة الشيفرة بأكملها، والترحيلات الكبيرة، والأبحاث المتقاطعة التحقق. شغّل /deep-research لرؤية السكربت المُضمَّن، أو /workflows لمتابعة عمليات التشغيل. ماذا يعني هذا: يمكنك الآن تولّي مهام بحجم الأسطول لا يمكن لمحادثة واحدة أن تنسّقها أبداً، مع حفظ التنسيق كأمر قابل لإعادة التشغيل. راجع سير العمل الديناميكي وultracode.
    آخر تحقّق: 2026-06-24. يتطلب Claude Code v2.1.154+؛ متاح على الخطط المدفوعة، وواجهة الـ API، وBedrock، وVertex، وFoundry. تأكّد من المصدر الرسمي.
  • 🧠 النماذج — أُطلق نموذج جديد من الفئة العليا، Claude Fable 5، في يونيو 2026 (Claude Code v2.1.170+). ماذا يعني هذا: تحقق من جدول النماذج والأسعار الحالية للاطلاع على التشكيلة الحيّة والمعرّفات الدقيقة قبل تثبيت نموذج — لا تُثبّت أبداً حقائق النماذج من تدوينة.
  • 🛠️ Claude Code — يمكن للوكلاء الفرعيين الآن توليد وكلائهم الفرعيين الخاصين (حتى 5 مستويات عمقاً)، ويطبّق /code-review --fix نتائج المراجعة على شجرة عملك. ماذا يعني هذا: يتركّب التفويض بعمق أكبر، ويمكن للمراجعة أن تُغلق الحلقة بدلاً من مجرد الإبلاغ. تحقق من سجل التغييرات الرسمي لمعرفة السلوك الدقيق.
  • 🔌 واجهة الـ API — تتطور قدرات مثل المخرجات المُهيكلة والتفكير الموسّع وتخزين الإيعازات مؤقتاً. ماذا يعني هذا: أعِد التحقق من قسم الـ API قبل الاعتماد على معامِلات دقيقة.
