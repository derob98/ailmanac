学習パス（自分のトラックを選ぶ）

⚪すべてのレベル

自分に最も近いと感じるトラックを選びましょう。それぞれが順序立てられたパスです。ステップは順番に進め、すでに知っていることは飛ばして構いません。所要時間はおおよその目安です。

🎯What you'll learn
  • ✅自分の目標に合う唯一のトラックを見つける — 初心者、ナレッジワーカー、ビルダー、チームリーダーのいずれか
  • ✅どこから始めるか推測するのではなく、明確で順序立てられた読み進め方をたどる
  • ✅各トラックにかかるおおよその時間を把握する（約45分〜約半日）
  • ✅どのトラックもしっくりこない場合にどこから始めるか

:::tip どのトラックが自分に合うか分からない？ 役職ではなく目標で選びましょう。コーディングせずに成果を出したい？ 初心者。実際の仕事をもっと速く？ ナレッジワーカー。Claude Code や API をカスタマイズしたい？ ビルダー。AI を他の人たちに展開したい？ チームリーダー。 :::

🌱 好奇心旺盛な初心者 · 約45分​

プログラマーにならずに AI を上手に使いたい人向け。

Guided walkthrough1 of 5
  1. 初級。5分で Claude から実際の成果を得る。/docs/start-here/your-first-5-minutes
  2. 初級。やりたいことに合ったモデルを選ぶ。/docs/start-here/which-claude
  3. 初級。回答を良くするための核となる動き。/docs/prompting/basics
  4. 初級。Claude の仕組みを平易な言葉で理解するメンタルモデル。/docs/foundations
  5. 初級。日々のタスクと小さな成功に応用する。/docs/playbooks/productivity

💼 ナレッジワーカー · 約2時間​

ライティング、リサーチ、分析など、実際の仕事を AI で速くしたい人向け。

Guided walkthrough1 of 5
  1. まず上の約45分のトラックを終えてから、以下に進む。
  2. 全レベル。実務にそのまま使える再利用可能なパターン。/docs/prompting
  3. 中級。本当に信頼できる出力を作る。/docs/foundations
  4. ライティング、リサーチ、データ分析から選ぶ。/docs/playbooks
  5. 永続的なワークスペース（/docs/claude-app）と、繰り返し使えるテンプレート（/docs/templates）。

🛠️ ビルダー / 開発者 · 約半日​

コードを書き、Claude Code をカスタマイズして API 上で構築したい人向け。

Guided walkthrough1 of 6
  1. 初級。CLI とは何か、そしてなぜ重要なのか。/docs/claude-code/what-is-claude-code
  2. 初級。Claude に自分のプロジェクトを教え、行動する前に計画させる。/docs/claude-code
  3. 中級。Claude にできることと振る舞いを制御する。/docs/claude-code
  4. 上級。1回の呼び出しからツール使用、完全なエージェントへと進む。/docs/api/first-call から始めて、次に /docs/api
  5. 中級。実際のプロジェクトですべてを組み合わせる。/docs/walkthroughs
  6. 上級。本番で構築するなら安全に出荷する。/docs/security

👥 チームリーダー · 約3時間​

AI をチームに展開しようとしており、一貫性、安全性、コストを重視する人向け。

Guided walkthrough1 of 5
  1. ここを続ける前に、上のビルダーの基本を押さえる。
  2. 上級。共有ツールキットを出荷し、チームが同じやり方で作業できるようにする。/docs/claude-code
  3. 中級。速度、品質、コストのバランスを取る。/docs/foundations と /docs/api/choosing-a-model
  4. 上級。自律的な実行を大規模で安全にする。/docs/security
  5. 全レベル。チームが守るべき規範を定める。/docs/security

理解度チェック

0/3
  1. 1コーディングはしないが、ライティングやリサーチを AI で速くしたい。どのトラックが合う？
  2. 2好奇心旺盛な初心者トラックはおおよそどのくらいの長さ？
  3. 3ビルダー / 開発者トラックの最初のステップは何？
💡Key takeaways
  • ⭐4つのトラック、ルールは1つ：目標で選び、ステップを順番にたどる。
  • ⭐所要時間は約45分（初心者）から約半日（ビルダー）まで。
  • ⭐トラックは入れ子になっている — ナレッジワーカーは初心者の上に、チームリーダーはビルダーの上に積み重なる。
  • ⭐迷ったら「最初の5分」から始めよう — どのトラックでも役に立つ。

どれが合うか分からない？ 最初の5分 から始めましょう。最終的にどこにたどり着くにせよ役に立ちます。そして レベルと鮮度の仕組み を読めば、バッジと日付の意味が分かるようになります。
