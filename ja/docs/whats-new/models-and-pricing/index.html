現行モデルと価格

⚪すべてのレベル

これは AILmanac の中で唯一、厳格な鮮度保証を持つページ です。サイト内の他の場所にあるモデル情報は、名前や数値をハードコーディングするのではなく、すべてこのページにリンクしています。そのため、最新に保つべき場所が 1 か所に集約されています。

Pricing, context-window sizes and rate limits change frequently and are intentionally NOT listed here — always read them on the official page above. This file is the single source of truth: never hard-code model facts in prose, link to this table instead.

価格やコンテキスト上限を表に含めていない理由​

これらは変更頻度が高すぎて安全にコピーできません — 古い価格は、価格がないことよりも悪いのです。各モデルの アイデンティティ（名前、API ID、用途）を一覧化し、価格、コンテキストウィンドウ、レート制限 については常に信頼できる公式ページへ案内します。

このページを最新に保つ方法​

  • 表は 「最終検証日」 を持つ単一のデータファイル（data/models.json）からレンダリングされます。
  • 更新は 1 行の変更だけ — 絶好の 最初の貢献 です。
  • 後のフェーズでは、自動化によりこのファイルを公式モデルエンドポイントと差分比較し、ずれが生じた際にプルリクエストを開く予定です — ただし、リリース前には必ず人間がレビューします。（私たちの自動化の仕組み を参照。）

モデルの選び方​

簡単な目安です（詳しいガイダンスは モデルの選択 を参照）:

  • まず Sonnet から。 ほとんどの作業において、品質・速度・コストのバランスが最も優れています。
  • 難しい問題でコストよりも品質が重要な場合は Opus を検討しましょう。
  • 大量処理、レイテンシ重視、または単純なタスクには Haiku に下げましょう。

  • 非推奨と移行ウォッチ — 廃止されたモデルと破壊的変更
