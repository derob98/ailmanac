Dein erster API-Aufruf

🟢Einsteiger
✅
Zuletzt verifiziert: 2026-06-20.

Die unten gezeigte API-Struktur ist stabil, aber Modell-IDs und SDK-Versionen ändern sich. Siehe die aktuelle Modelltabelle.

Bestätige anhand der offiziellen Quelle.

Die API ermöglicht es deinem Programm, mit Claude zu sprechen. Am Ende dieser Lektion hast du eine echte Anfrage ausgeführt und verstehst genau, was jeder Bestandteil davon tut.

🎯What you'll learn
  • ✅Einen API-Schlüssel erstellen und sicher als Umgebungsvariable speichern
  • ✅Das offizielle SDK für Python oder TypeScript installieren (oder einfaches cURL verwenden)
  • ✅Deine erste messages-Anfrage senden und Claudes Antwort lesen
  • ✅Die vier Kernfelder verstehen: model, max_tokens, messages und system

Der gesamte Weg in drei Schritten​

Alles unten passt in eine einfache Abfolge. Behalte diese Karte im Kopf, während du vorgehst.

Guided walkthrough1 of 3
  1. Erstelle einen Schlüssel in der Anthropic Console und exportiere ihn als Umgebungsvariable, damit er niemals in deinem Code liegt.
  2. Füge das anthropic-Paket für Python oder @anthropic-ai/sdk für TypeScript hinzu. Bei cURL gibt es nichts zu installieren.
  3. Sende eine Liste von Nachrichten an das Modell und gib den zurückgegebenen content aus.

1. Einen API-Schlüssel besorgen​

Erstelle einen in der Anthropic Console. Setze ihn dann als Umgebungsvariable, damit er niemals in deinem Code liegt:

✦

Exportiere deinen API-Schlüssel

export ANTHROPIC_API_KEY="sk-ant-..."
⚠️Watch out
  • •Committe niemals deinen Schlüssel. Bewahre Schlüssel in Umgebungsvariablen oder einem Secrets-Manager auf, niemals in der Versionsverwaltung. Siehe die Sicherheitsseite (/docs/security).

2. Das SDK installieren​

Wähle deine Sprache. Das SDK liest den Schlüssel automatisch aus der Umgebungsvariable, die du gerade gesetzt hast.

  • Python
  • TypeScript
  • cURL
✦

Installation (Python)

pip install anthropic
✦

Installation (TypeScript)

npm install @anthropic-ai/sdk

Nichts zu installieren — du brauchst nur curl.

3. Den Aufruf ausführen​

Jede Anfrage ist eine Liste von messages. Das Modell antwortet mit content. Führe den Codeausschnitt für deine Sprache aus, und du wirst sehen, wie Claude die Frage beantwortet.

  • Python
  • TypeScript
  • cURL
import anthropic

client = anthropic.Anthropic()  # reads ANTHROPIC_API_KEY from the environment

message = client.messages.create(
    model="claude-sonnet-5",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "In one sentence, what is an API?"}
    ],
)

print(message.content[0].text)
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic(); // reads ANTHROPIC_API_KEY from the environment

const message = await client.messages.create({
  model: "claude-sonnet-5",
  max_tokens: 1024,
  messages: [
    { role: "user", content: "In one sentence, what is an API?" },
  ],
});

console.log(message.content[0].text);
curl https://api.anthropic.com/v1/messages \
  --header "x-api-key: $ANTHROPIC_API_KEY" \
  --header "anthropic-version: 2023-06-01" \
  --header "content-type: application/json" \
  --data '{
    "model": "claude-sonnet-5",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "In one sentence, what is an API?"}
    ]
  }'

Was gerade passiert ist​

Du hast vier Informationen gesendet. Hier ist, was jede davon steuert:

  • model — welches Claude verwendet werden soll. Verdrahte es nicht blind fest; siehe Ein Modell auswählen.
  • max_tokens — eine Obergrenze für die Antwortlänge (in Tokens). Es legt nicht das Kontextfenster fest.
  • messages — die bisherige Konversation. Die API ist zustandslos: Um einen Chat fortzusetzen, sende jedes Mal den gesamten Verlauf zurück.
  • system (optional) — eine übergeordnete Anweisung, die Claudes Rolle für den Aufruf festlegt.

✨Pro tip
  • →Da die API zustandslos ist, ist ein mehrstufiger Chat einfach ein wachsendes messages-Array: hänge jede Antwort an und sende beim nächsten Aufruf die vollständige Liste erneut.

💡Key takeaways
  • ⭐Drei Schritte: einen Schlüssel besorgen, das SDK installieren, eine messages-Anfrage senden.
  • ⭐Speichere den Schlüssel in ANTHROPIC_API_KEY, damit das SDK ihn automatisch liest und er niemals in deinen Code gelangt.
  • ⭐Eine Anfrage ist eine Liste von Nachrichten; die Antwort kommt in content zurück.
  • ⭐Die API ist zustandslos: sende den vollständigen Konversationsverlauf erneut, um einen Chat fortzusetzen.
  • ⭐model, max_tokens, messages und system sind die Felder, zu denen du zuerst greifen wirst.
