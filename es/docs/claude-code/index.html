Claude Code

El núcleo profundo y autorizado sobre la herramienta agéntica de CLI/IDE: todas las superficies configurables, organizadas por nivel, desde CLAUDE.md hasta los agentes y el SDK.

QQué es Claude Code

La herramienta agéntica CLI/IDE explicada — el bucle de sesión, la instalación y dónde vive cada cosa.

CCLAUDE.md y archivos de memoria

La personalización de Claude Code de mayor impacto: memoria persistente del proyecto.

AAGENTS.md e interoperabilidad entre herramientas

El estándar abierto que lee todo agente de programación, y cómo mantenerlo sincronizado con tu CLAUDE.md.

MModo Plan

Investigar y proponer en modo de solo lectura antes de que se ejecute cualquier edición: la forma más segura de abordar cambios grandes.

IIntegraciones con IDE

Extensiones de VS Code y JetBrains, diffs en línea y ajustes compartidos.

Ssettings.json: El sistema de configuración

Niveles de archivos y precedencia, secciones clave y configuración de proyecto frente a personal.

PPermisos y modos de permiso

Reglas allow/ask/deny y modos: reduce las preguntas sin perder seguridad.

CComandos slash personalizados

Convierte un flujo de trabajo repetible en una sola palabra.

EEstilos de salida

Cambia el tono, el nivel de detalle y el comportamiento didáctico de Claude Code.

PPersonalización de la statusline

Muestra el modelo, el cwd, el presupuesto de contexto y el estado de git con un script personalizado.

DDesarrollo guiado por especificaciones con Spec Kit

Deja de programar a ojo las funciones grandes. Convierte una especificación ejecutable en la fuente de verdad y deja que Claude Code construya a partir de ella — con Spec Kit de GitHub.

GGestión del contexto

/compact frente a /clear, compactación automática y cómo mantener la eficacia a medida que crecen las sesiones.

CCheckpoints y Rewind

Deshaz las ediciones de Claude, revierte una sesión entera y comprime el contexto con /rewind — la red de seguridad para ejecuciones ambiciosas.

HHooks: automatización determinista

Ejecuta comandos en eventos del ciclo de vida para imponer formato, controles y seguridad.

SSkills: Experiencia bajo demanda

SKILL.md, divulgación progresiva y Skill frente a comando frente a subagente frente a MCP.

SSubagentes y agentes paralelos

Delega en agentes de contexto aislado con herramientas acotadas — y cuándo no hacerlo.

FFlujos de trabajo dinámicos y ultracode

Orquesta de decenas a cientos de subagentes desde un script que Claude escribe, para auditorías de bases de código, grandes migraciones e investigación con verificación cruzada.

SServidores MCP en Claude Code

Conecta bases de datos, GitHub, navegadores y mucho más mediante el Model Context Protocol.

TTareas en segundo plano, /loop y programación

Comandos de larga duración, bucles recurrentes y agentes en la nube programados.

PPlugins y marketplaces

Agrupa skills, agentes, comandos, hooks y MCP en una única unidad instalable.

GGit Worktrees y flujos de trabajo paralelos

Ejecuta varias sesiones de Claude aisladas sobre un mismo repo sin que colisionen entre sí.

MModo Headless y el Agent SDK

Ejecuta Claude Code en CI y crea agentes de producción sobre el mismo harness.

TThe MCP Token Tax

Why connecting MCP servers can burn 50k–150k tokens before your first request — and the three fixes: Tool Search, code execution, and knowing when a CLI call wins.

AAccessibility

Screen reader mode, vim ergonomics, mouse and scroll controls, and high-contrast themes in Claude Code.

TTroubleshooting Claude Code

Symptom-to-fix for hangs, high memory, broken search, compaction thrashing, and garbled terminals.
