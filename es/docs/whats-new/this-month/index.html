Este mes en Claude

⚪Todos los niveles
✅
Última verificación: 2026-06-24.

Esta página es un resumen continuo y curado por personas. Para la lista autorizada y completa de cambios, consulta siempre la documentación oficial de Anthropic y el registro de cambios de Claude Code.

Confirma con la fuente oficial.

Un resumen en lenguaje claro de lo que ha cambiado recientemente en Claude — para que puedas ojear "qué hay de nuevo y qué significa para mí" sin leer todos los registros de cambios. Lo más reciente primero.

:::info Cómo se mantiene actualizada esta página Las entradas las añaden la comunidad y un vigilante automatizado que detecta cambios upstream para que una persona los redacte. Cada entrada enlaza su fuente oficial. ¿Echas algo en falta? Añádelo. :::

Cómo leer una entrada​

Cada actualización está etiquetada por superficie — 🗣️ Aplicaciones (chat/voz/móvil), 🛠️ Claude Code, 🔌 API, 🧠 Modelos — y responde a "qué significa para ti."

Actualizaciones recientes​

  • 🧠 Modelos — Claude Sonnet 5 se publicó el 30 de junio de 2026 y ahora es el modelo predeterminado en Claude Code, con una ventana de contexto nativa de 1M-token y precios introductorios hasta el 31 de agosto de 2026. Es el nuevo nivel Sonnet equilibrado de "empieza aquí", que reemplaza a Claude Sonnet 4.6 (ahora un modelo heredado). Qué significa: si fijas un modelo, migra a claude-sonnet-5; y consulta siempre la tabla Modelos y precios actuales para conocer los IDs exactos y los precios en vivo antes de depender de un número sacado de una entrada de blog.
    ✅
    Última verificación: 2026-07-01. ID de API claude-sonnet-5; nivel Sonnet; contexto de 1M-token; salida máxima de 128k; corte de conocimiento en enero de 2026. Los precios introductorios de $2 / $10 por MTok se aplican hasta el 31 de agosto de 2026 (estándar $3 / $15). Ahora es el predeterminado de Claude Code según el registro de cambios (v2.1.197, June 30 2026). Confirma con la fuente oficial.
  • 🛠️ Claude Code — Los flujos de trabajo dinámicos están en vista previa de investigación: solicita uno (o configura /effort ultracode) y Claude escribe un script de JavaScript que orquesta de decenas a cientos de subagentes en segundo plano — para auditorías de todo el código, migraciones grandes e investigación con verificación cruzada. Ejecuta /deep-research para ver el que viene incluido, o /workflows para observar las ejecuciones. Qué significa: ahora puedes asumir tareas a escala de flota que una sola conversación nunca podría coordinar, con la orquestación guardada como un comando reejecutable. Consulta Flujos de trabajo dinámicos y ultracode.
    ✅
    Última verificación: 2026-06-24. Requiere Claude Code v2.1.154+; disponible en los planes de pago, la API, Bedrock, Vertex y Foundry. Confirma con la fuente oficial.
  • 🧠 Modelos — Un nuevo modelo de primer nivel, Claude Fable 5, se publicó en junio de 2026 (Claude Code v2.1.170+). Qué significa: consulta la tabla Modelos y precios actuales para ver la lista en vivo y los IDs exactos antes de fijar un modelo — nunca codifiques datos de modelos sacados de una entrada de blog.
  • 🛠️ Claude Code — Los subagentes ahora pueden generar sus propios subagentes (hasta 5 niveles de profundidad), y /code-review --fix aplica los hallazgos de la revisión a tu árbol de trabajo. Qué significa: la delegación se compone más profundamente, y la revisión puede cerrar el ciclo en lugar de solo informar. Contrasta con el registro de cambios oficial para conocer el comportamiento exacto.
  • 🔌 API — Capacidades como la salida estructurada, el pensamiento extendido y el almacenamiento en caché de prompts evolucionan. Qué significa: vuelve a comprobar la sección de la API antes de depender de parámetros exactos.
