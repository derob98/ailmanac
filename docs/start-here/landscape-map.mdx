AILmanac doesn't exist in a vacuum, and we won't pretend to replace everything. This lesson is your honest map of the best resources in the ecosystem — and, just as importantly, when to use them **instead of** us.

<Callout type="objectives" items={["Identify the single source of truth and why the docs always win when they disagree with us", "Recognize the official Anthropic resources and what each one is for", "Know the best community and vendor-neutral prompting resources", "Match any need to the right destination using a simple decision table", "Place AILmanac in the landscape as the opinionated field guide"]} />

## The source of truth

Start here, because everything else orbits this one rule.

<Callout type="warning" items={["When AILmanac and the official docs disagree, the docs win. Use them for exact, current specifics; use us for opinionated, layered, learn-by-doing guidance."]} />

- **[Anthropic Documentation](https://platform.claude.com/docs)** — the authoritative reference for Claude, the API, models, and features.

## Official Anthropic resources

These are first-party. Reach for them when you want canonical examples, runnable code, or guided learning straight from the source.

<Flashcards title="Official resources at a glance" cards={[{front: "Prompt Library", back: "Ready-made example prompts."}, {front: "Anthropic Cookbook", back: "Runnable API recipes — tool use, RAG, vision."}, {front: "Claude Code docs", back: "The CLI/IDE reference."}, {front: "Anthropic Academy / Learn", back: "Free guided courses."}, {front: "Model Context Protocol", back: "The open MCP standard plus server directory."}]} />

- **[Prompt Library](https://platform.claude.com/docs/en/resources/prompt-library/library)** — ready-made example prompts.
- **[Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)** — runnable API recipes (tool use, RAG, vision).
- **[Claude Code docs](https://code.claude.com/docs/en/overview)** — the CLI/IDE reference.
- **[Anthropic Academy / Learn](https://www.anthropic.com/learn)** — free guided courses.
- **[Model Context Protocol](https://modelcontextprotocol.io)** — the open MCP standard + server directory.

## Community & general

When you want vendor-neutral depth or curated, community-tested workflows, these are the standouts.

- **[awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code)** — curated commands, hooks, and workflows.
- **[Prompt Engineering Guide](https://www.promptingguide.ai)** — deep, vendor-neutral prompting reference.
- **[Learn Prompting](https://learnprompting.org)** — structured, beginner-friendly prompting course.

## When to use what

The fastest way to use this map: start from what you need, then jump to the matching destination.

|---|---|

| Copy-paste example prompts | Prompt Library / our [templates](/docs/templates/prompts) |
| Runnable API code | Cookbook |
| An opinionated, level-tagged path | **AILmanac** |

| A guided course | Anthropic Academy |

## Where AILmanac fits

We're the **opinionated field guide**: one recommended way first, tagged by level, template-rich, honest about freshness, and spanning Claude **and** general AI concepts. Think of us as the map that points you to the right deep resource at the right time — and teaches the fundamentals so they all make sense.
