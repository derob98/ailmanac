AILmanac doesn't exist in a vacuum, and we won't pretend to replace everything. Here's an honest map of the best resources — and when to use them **instead of** us.

## The source of truth

- **[Anthropic Documentation](https://docs.anthropic.com)** — the authoritative reference for Claude, the API, models, and features. **When AILmanac and the docs disagree, the docs win.** Use them for exact, current specifics; use us for opinionated, layered, learn-by-doing guidance.

## Official Anthropic resources

- **[Prompt Library](https://docs.anthropic.com/en/resources/prompt-library/library)** — ready-made example prompts.
- **[Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)** — runnable API recipes (tool use, RAG, vision).
- **[Claude Code docs](https://code.claude.com/docs/en/overview)** — the CLI/IDE reference.
- **[Anthropic Academy / Learn](https://www.anthropic.com/learn)** — free guided courses.
- **[Model Context Protocol](https://modelcontextprotocol.io)** — the open MCP standard + server directory.

## Community & general

- **[awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code)** — curated commands, hooks, and workflows.
- **[Prompt Engineering Guide](https://www.promptingguide.ai)** — deep, vendor-neutral prompting reference.
- **[Learn Prompting](https://learnprompting.org)** — structured, beginner-friendly prompting course.

## When to use what

| You want… | Go to |
|---|---|
| Exact current API/feature details | Official docs |
| Copy-paste example prompts | Prompt Library / our [templates](/docs/templates/prompts) |
| Runnable API code | Cookbook |
| An opinionated, level-tagged path | **AILmanac** |
| A deep theory reference on prompting | promptingguide.ai |
| A guided course | Anthropic Academy |

## Where AILmanac fits

We're the **opinionated field guide**: one recommended way first, tagged by level, template-rich, honest about freshness, and spanning Claude **and** general AI concepts. Think of us as the map that points you to the right deep resource at the right time — and teaches the fundamentals so they all make sense.
