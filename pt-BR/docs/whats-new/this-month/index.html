Este Mês no Claude

Todos os níveis
Última verificação: 2026-06-24.

Esta página é um resumo contínuo, curado por humanos. Para a lista oficial e completa de mudanças, consulte sempre a documentação oficial da Anthropic e o changelog do Claude Code.

Confirme na fonte oficial.

Um resumo em linguagem simples do que mudou recentemente em todo o Claude — para que você possa dar uma olhada rápida em "o que há de novo e o que isso significa para mim" sem ler cada changelog. Do mais recente ao mais antigo.

:::info Como esta página se mantém atualizada As entradas são adicionadas pela comunidade e por um observador automatizado que sinaliza mudanças upstream para que um humano as documente. Cada entrada aponta para sua fonte oficial. Notou algo faltando? Adicione. :::

Como ler uma entrada​

Cada atualização é etiquetada por superfície — 🗣️ Apps (chat/voz/mobile), 🛠️ Claude Code, 🔌 API, 🧠 Modelos — e responde "o que isso significa para você."

Atualizações recentes​

  • 🧠 Modelos — O Claude Sonnet 5 foi lançado em 30 de junho de 2026 e agora é o modelo padrão no Claude Code, com uma janela de contexto nativa de 1M-token e preços introdutórios até 31 de agosto de 2026. É o novo nível Sonnet equilibrado "comece por aqui", substituindo o Claude Sonnet 4.6 (agora um modelo legado). O que isso significa: se você fixa um modelo, migre para claude-sonnet-5; e sempre confira a tabela de Modelos e Preços Atuais para os IDs exatos e preços ao vivo antes de depender de um número visto em um post de blog.
    Última verificação: 2026-07-01. ID de API claude-sonnet-5; nível Sonnet; contexto de 1M-token; saída máxima de 128k; corte de conhecimento em Jan 2026. Preços introdutórios de $2 / $10 por MTok aplicam-se até Aug 31, 2026 (padrão $3 / $15). Agora é o padrão do Claude Code conforme o changelog (v2.1.197, June 30 2026). Confirme na fonte oficial.
  • 🛠️ Claude Code — Os fluxos de trabalho dinâmicos estão em research preview: peça um (ou defina /effort ultracode) e o Claude escreve um script JavaScript que orquestra dezenas a centenas de subagentes em segundo plano — para auditorias em toda a base de código, grandes migrações e pesquisa com verificação cruzada. Execute /deep-research para ver o que vem incluído, ou /workflows para acompanhar as execuções. O que isso significa: agora você pode assumir tarefas em escala de frota que uma única conversa jamais conseguiria coordenar, com a orquestração salva como um comando reexecutável. Veja Fluxos de Trabalho Dinâmicos e ultracode.
    Última verificação: 2026-06-24. Requer Claude Code v2.1.154+; disponível em planos pagos, na API, no Bedrock, no Vertex e no Foundry. Confirme na fonte oficial.
  • 🧠 Modelos — Um novo modelo de nível superior, o Claude Fable 5, foi lançado em junho de 2026 (Claude Code v2.1.170+). O que isso significa: confira a tabela de Modelos e Preços Atuais para a linha de modelos ao vivo e os IDs exatos antes de fixar um modelo — nunca codifique fatos de modelo a partir de um post de blog.
  • 🛠️ Claude Code — Os subagentes agora podem gerar seus próprios subagentes (até 5 níveis de profundidade), e /code-review --fix aplica os resultados da revisão à sua working tree. O que isso significa: a delegação se compõe mais fundo, e a revisão pode fechar o ciclo em vez de apenas relatar. Confira o changelog oficial para o comportamento exato.
  • 🔌 API — Capacidades como saída estruturada, extended thinking e prompt caching evoluem. O que isso significa: verifique novamente a seção da API antes de depender de parâmetros exatos.
