Claude Code

O núcleo profundo e definitivo sobre a ferramenta agêntica de CLI/IDE — cada ponto configurável, organizado por nível, do CLAUDE.md aos agentes e ao SDK.

OO Que É o Claude Code

A ferramenta CLI/IDE agêntica explicada — o loop de sessão, instalação e onde tudo fica.

CCLAUDE.md e arquivos de memória

A personalização de maior alavancagem do Claude Code: memória persistente do projeto.

AAGENTS.md & Interoperabilidade Entre Ferramentas

O padrão aberto que todo agente de código lê — e como mantê-lo sincronizado com seu CLAUDE.md.

MModo Plano

Investigar e propor em modo somente leitura antes que qualquer edição seja executada — a forma mais segura de encarar grandes mudanças.

IIntegrações com IDE

Extensões para VS Code e JetBrains, diffs inline e configurações compartilhadas.

Ssettings.json: O Sistema de Configuração

Camadas de arquivos e precedência, seções principais e configuração de projeto vs pessoal.

PPermissões e Modos de Permissão

Regras allow/ask/deny e modos — reduza prompts sem perder segurança.

CComandos Slash Personalizados

Transforme um fluxo de trabalho repetível em uma única palavra.

EEstilos de Saída

Mude o tom, a verbosidade e o comportamento didático do Claude Code.

PPersonalização da Statusline

Exiba o modelo, o cwd, o orçamento de contexto e o estado do git com um script personalizado.

DDesenvolvimento Orientado a Especificações com Spec Kit

Pare de fazer vibe-coding em features grandes. Faça de uma especificação executável a fonte da verdade e deixe o Claude Code construir a partir dela — com o Spec Kit do GitHub.

GGerenciamento de Contexto

/compact vs /clear, compactação automática e como manter a eficácia conforme as sessões crescem.

HHooks: Automação Determinística

Execute comandos em eventos do ciclo de vida para impor formatação, barreiras e segurança.

SSkills: Expertise Sob Demanda

SKILL.md, divulgação progressiva e Skill vs comando vs subagente vs MCP.

SSubagentes e Agentes Paralelos

Delegue a agentes de contexto isolado com ferramentas restritas — e quando não fazer isso.

WWorkflows dinâmicos e ultracode

Orquestre de dezenas a centenas de subagentes a partir de um script que o Claude escreve — para auditorias de base de código, grandes migrações e pesquisas verificadas de forma cruzada.

SServidores MCP no Claude Code

Conecte bancos de dados, GitHub, navegadores e muito mais via Model Context Protocol.

TTarefas em Segundo Plano, /loop e Agendamento

Comandos de longa duração, loops recorrentes e agentes na nuvem agendados.

PPlugins e Marketplaces

Agrupe skills, agentes, comandos, hooks e MCP em uma única unidade instalável.

GGit Worktrees e Fluxos de Trabalho Paralelos

Execute várias sessões isoladas do Claude em um único repositório sem que elas colidam.

MModo Headless e o Agent SDK

Execute o Claude Code em CI e construa agentes de produção sobre o mesmo harness.
